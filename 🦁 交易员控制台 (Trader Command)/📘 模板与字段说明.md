# 📘 模板与字段说明（Trader Command v5）

## 1) 封面图（`封面/cover`）自动化

- 三个模板都支持：如果 frontmatter 的 `封面/cover` 为空，会从正文锚点 `<!--PA_COVER_SOURCE-->` 下方提取**第一张图片**并自动写入 `封面/cover`。
- 写入格式：优先写成 `![[path/to/image.png]]`（与画廊/预览解析一致）。
- 不会覆盖：只在 `封面/cover`（或 `cover`）为空时才写入。

**使用方式（最省事）**
1) 在 `<!--PA_COVER_SOURCE-->` 下面粘贴截图/图表
2) 等待 DataviewJS 渲染一次（阅读模式或重新打开/刷新）
3) 你会看到 frontmatter 的 `封面/cover` 自动出现

## 2) Trade Note：自动计算

- `入场/entry_price` + `止损/stop_loss` + `目标位/take_profit` 填在属性里
- 正文会自动输出：Risk / Reward / R（缺字段时显示占位）

## 3) SR / 卡片（可选）

- 只有给笔记加 `#flashcards` 才会进入复习统计
- 支持语法：
  - `问题 :: 答案`
  - `答案 ::: 问题`
  - `==cloze==`

（模板里示例放在代码块里，不会被计入；你要制卡就写在普通正文行）
