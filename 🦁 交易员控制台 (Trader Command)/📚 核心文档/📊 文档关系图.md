## 使用指南

### 场景1: 新窗口AI启动

```
1. 打开 README.md
2. 阅读 📚 项目技术文档-完整架构手册.md
3. 5分钟内恢复完整上下文
```

### 场景2: 开始新任务

```
1. 查看 📋 插件功能优化-主任务列表.md 确定当前Phase
2. 参考 🎯 插件优化项目-目标架构.md 确认目标
3. 遵循 🧠 插件智能优化技能.md 的标准流程
```

### 场景3: 遇到问题

```
1. 查看 🧠 插件智能优化技能.md 的失败教训
2. 参考成功经验寻找解决方案
3. 更新经验库
```

### 场景4: 查看历史

```
1. 进入 📂 存档/ 文件夹
2. 查看相关历史报告
```
