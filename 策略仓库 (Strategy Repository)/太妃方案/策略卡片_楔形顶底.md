# 🎯 策略概览

**一句话描述**: 
对趋势极值两次突破皆失败,形成三推收敛结构,预示反转的高概率反转形态。

**Al Brooks术语**: Wedge Top/Bottom, Three Pushes Pattern  
**太妃原始术语**: 楔形顶底、先行顶底

---

## 📋 入场条件 (Entry Criteria)

### 市场背景要求
- [x] 市场周期: **强趋势**(末端) 或 **弱趋势/通道**
- [x] 波段结构: 三次同向推动,一推不如一推
- [x] 动能特征: 收敛外观,推动屡屡减弱

### 关键识别要素
1. **楔形定义**:
   * 三推结构(Three Pushes): 向旗形倾斜方向产生三次同向推动
   * 突破失败: 对趋势极值两次突破皆失败
   * 均衡暗示: 突破失败表明均衡应不在该方向上
2. **收敛要求**:
   * 可以收敛、扩张、倾斜或几乎水平
   * 通常不断递进,但第二、三推也可过冲或不及
   * 外观不是重点,关键是**逆势力量不济**
3. **与双重顶底区别**:
   * 楔形: 三推结构
   * 双重顶底: 只有二推

### 信号棒特征
- [x] 位置: 第三推处(或第二推不及第一推时)
- [x] K线形态: 反转棒(强收盘在相反方向)
- [x] 确认: 突破楔形趋势线
- [x] 理想: 第三推过冲或持平第二推(更易识别)

### 特殊类型识别
1. **头肩顶底**: 第三推不及第二推极值
2. **截断楔形**: 无法递进的第三推
3. **缩短楔形**: 第二推不及第一推

---

## 🛡️ 风险管理 (Risk Management)

### 止损位置
* **标准**: 第三推极值外2-4跳
  * 牛市楔形顶: 第三推高点上方
  * 熊市楔形底: 第三推低点下方
* **保守**: 楔形起点外(第一推极值外)
  * 适用于资金量较大的交易者

### 目标位计算
* **首要目标**: 楔形起点(第一推的起始位置)
* **次要目标**: 等距投射
  * 楔形高度(第一推到第三推的垂直距离)
  * 从突破点向反方向投射
* **延伸目标**: 如反转成功,可持有至更大级别支撑/阻力

### 入场时机选择
* **激进**: 第三推处直接反向入场
* **保守**: 等待突破楔形趋势线后入场
* **最保守**: 等待重测楔形趋势线不破后入场(High 2/Low 2)

---

## ⚠️ 失败模式 (Failure Signals)

当出现以下情况,立即退出或不入场:
- [x] **第四推出现**: 楔形延伸为旗形,转为持续形态
- [x] **突破后无跟进**: 突破趋势线后立即反向
- [x] **横盘整理**: 楔形内部形成交易区间
- [x] **动能增强**: 第三推比第二推更强劲(不符合收敛要求)
- [x] **嵌于强趋势**: 在强趋势中途的楔形倾向引发趋势恢复而非反转

### 楔形vs旗形的判断
* **楔形**: 三推后反转(反转形态)
* **旗形**: 三推后突破继续原趋势(持续形态)
* **关键**: 嵌于趋势中时,倾向引发趋势恢复(成为旗形)

## 📊 Al Brooks理论对应

### 核心概念映射
| 太妃术语 | Al Brooks术语 | 说明 |
|---------|--------------|------|
| 楔形顶底 | Wedge Top/Bottom | 三推收敛反转形态 |
| 三推结构 | Three Pushes | 三次同向推动 |
| 先行顶底 | Leading Top/Bottom | 逆势三推收敛 |
| 截断楔形 | Truncated Wedge | 第三推不及第二推 |
| 一推不如一推 | Weakening Pushes | 动能递减 |

### 适用的Al Brooks形态
- **Wedge Bull/Bear Flag**: 楔形旗(可反转可延续)
- **Three Pushes Up/Down**: 三推形态
- **Climax Reversal**: 高潮反转(急剧楔形)
- **Final Flag**: 末端旗形(楔形的变体)

## 🎓 学习笔记 (Notes)

### 课程来源
* **视频**: 太妃价格行为 L14B - 双重顶底-楔形
* **时间码**: 01:44 - 楔形顶底定义
* **时间码**: 02:46 - 如何交易楔形

### 关键要点
1. **本质特征**: 外观不是重点,关键是逆势力量不济,最终将反转自身转向原趋势
2. **收敛宽松**: 实际不必严格收敛,标准较宽松
3. **形态变体**: 可以二长三短,或二短三长
4. **形成机制**:
   * 顺势交易者的通道收敛是主因
   * 多头不愿等待更深回调就急于买进
   * 空头也不等待更深下跌就急于止盈
5. **市场意义**: 推动屡屡减弱表明价格向该方向突破意愿降低

### 个人理解
* 楔形是Al Brooks体系中最重要的反转形态之一
* 关键是"三推",如果只有两推就是双重顶底
* 第三推的表现决定交易质量: 过冲/持平>不及
* 在强趋势中途的楔形要小心,可能只是牛旗/熊旗

## 🔄 操作检查清单 (Checklist)

**入场前检查**:
- [ ] 数推动: 确认三次同向推动(Three Pushes)
- [ ] 观察收敛: 一推不如一推的迹象
- [ ] 第三推质量: 是过冲/持平/不及?
- [ ] 突破确认: 是否突破楔形趋势线?
- [ ] 位置评估: 是趋势末端还是中途?(末端更可靠)

**持仓中监控**:
- [ ] 观察是否向楔形起点运动
- [ ] 警惕第四推出现(转为旗形)
- [ ] 关注重测趋势线的表现

**复盘问题**:
- [ ] 三推识别是否正确?
- [ ] 是否过早入场(第二推就入场)?
- [ ] 止损设置合理吗?
- [ ] 是楔形还是旗形?(反转还是延续)

> [!tip] 实战建议
> 1. **新手**: 只交易第三推过冲或持平的楔形,更易识别
> 2. **中级**: 可交易截断楔形(第三推不及),但需更谨慎
> 3. **高手**: 区分趋势末端楔形(高胜率反转)vs中途楔形(可能延续)

> [!warning] 风险提示
> * 强趋势中途的楔形可能演变为旗形继续原趋势
> * 第四推出现时要警惕,可能转为持续形态
> * 过早入场(第二推)风险较大
> * 楔形内部如形成区间,反转概率降低
