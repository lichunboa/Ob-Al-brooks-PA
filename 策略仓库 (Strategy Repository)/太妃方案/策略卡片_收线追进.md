# 🎯 策略概览

**一句话描述**: 
在强劲突破后的急速/窄幅通道中,每根趋势K线收盘时市价追进,捕捉趋势动能延续。

**Al Brooks术语**: Buy/Sell NOW (Market Order Entry during Strong Trend)  
**太妃原始术语**: 收线市价追进方案、BUYNOW急速建仓方案

---

## 📋 入场条件 (Entry Criteria)

### 市场背景要求
- [x] 市场周期: **强趋势** 或 **突破模式**
- [x] 波段结构: 急速(Spike) 或 窄幅通道(Tight Channel)
- [x] 动能特征: 连续同向K线,无明显回调

### 关键识别要素
1. **急速定义**: 连续同向K线运行,低点不降低(牛市)/高点不抬高(熊市)
2. **窄幅通道**: 逆势方无法通过限价单盈利,动能保持良好
3. **等距运动**: 突破后通常伴随第二推,AB=CD或实体高度投射

### 信号棒特征
- [x] K线形态: **强趋势K线**(大阳线/大阴线)
- [x] 收盘位置: 强收盘,接近K线极值
- [x] 上下影线: 短影线或无影线
- [x] 重叠程度: 与前K线几乎无重叠

### 路径分析
- [ ] 突破点距离显著磁体(支撑/阻力)较远
- [ ] 前方无明显阻力障碍
- [ ] 跟进K线收线强劲

---

## 🛡️ 风险管理 (Risk Management)

### 止损位置
* **标准**: 推动起点之外(Swing Point Beyond Entry Bar)
  * 牛市: 突破前低点下方2-4跳
  * 熊市: 突破前高点上方2-4跳
* **激进**: 信号棒低点/高点外1-2跳

### 目标位计算
* **首要目标**: 等距投射(Measured Move)
  * 实体高度投射: 突破K线实体高度 × 1
  * AB=CD模式: 第一推幅度 = 第二推幅度
* **次要目标**: 通道上轨/下轨
* **延伸目标**: 如动能持续,可持有至更大级别磁体

### 仓位管理
* **学习阶段**: 最小仓位(1手),每根K线只追1次
* **验证阶段**: 可在前2-3根趋势K线分批追进
* **熟练阶段**: 根据动能强度决定是否加码

---

## ⚠️ 失败模式 (Failure Signals)

当出现以下情况,立即退出或不入场:
- [x] **急速结束**: 出现首根回调K线(低点降低/高点抬高)
- [x] **动能衰竭**: K线实体缩小,上下影线变长
- [x] **磁体临近**: 价格接近显著支撑/阻力位
- [x] **重叠增加**: 连续3根K线出现明显重叠(铁丝网 Barbwire)
- [x] **假突破**: 突破后立即反向收线

## 📊 Al Brooks理论对应

### 核心概念映射
| 太妃术语 | Al Brooks术语 | 说明 |
|---------|--------------|------|
| 急速 | Spike | 连续同向K线无回调 |
| 窄幅通道 | Tight Channel | 小幅回调但趋势保持 |
| 收线追进 | Buy/Sell NOW | 收盘价市价单入场 |
| 等距运动 | Measured Move (MM) | AB=CD或实体投射 |
| 推动起点 | Swing Point | 趋势开始的转折点 |

### 适用的Al Brooks形态
- **Spike and Channel**: 急速后转窄幅通道
- **Strong Breakout with Follow-Through**: 强突破+跟进
- **Trending Trading Range**: 趋势中的小区间突破

## 🎓 学习笔记 (Notes)

### 课程来源
* **视频**: 太妃价格行为 L13A - 急速交易 - 收线
* **时间码**: 13:13 - 收线市价追进方案

### 关键要点
1. **数学优势来源**: 趋势中80%以上的反转尝试会失败
2. **自我强化机制**: 突破→追进→延续形成正反馈循环
3. **等距运动规律**: 第一推完成后大概率出现第二推
4. **窄幅通道本质**: 更大时间级别上看就是急速运动

### 个人理解
* 这个策略最适合开盘时的强趋势(Trend from Open)
* 关键是识别"急速"阶段,一旦转入宽通道就不再适用
* 止损相对较大,但目标位明确,适合波段交易
* 需要克服"追高杀低"的心理障碍

## 🔄 操作检查清单 (Checklist)

**入场前检查**:
- [ ] 确认当前处于急速或窄幅通道阶段
- [ ] 当前K线即将收盘且为强趋势K线
- [ ] 前方无明显磁体阻力
- [ ] 推动起点明确,可设置止损

**持仓中监控**:
- [ ] 观察后续K线是否继续同向
- [ ] 如出现回调K线立即关注是否止损
- [ ] 接近目标位时考虑部分获利

**复盘问题**:
- [ ] 是否在最佳时机入场(收线时)?
- [ ] 止损是否设置合理?
- [ ] 是否过早离场错过目标位?

> [!tip] 实战建议
> 1. **新手**: 只在开盘第一个急速追1-2根K线,见好就收
> 2. **中级**: 可追3-5根K线,利用等距运动设定目标位
> 3. **高手**: 结合更大周期判断,决定是否持有过夜

> [!warning] 风险提示
> * 该策略盈亏比中等,依赖胜率,不适合频繁使用
> * 急速突然结束时止损较大,需严格执行
> * 避免在盘整末期或磁体附近使用
