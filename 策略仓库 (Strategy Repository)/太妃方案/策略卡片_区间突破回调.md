# 🎯 策略概览

**一句话描述**: 
区间突破成功后,等待价格回调测试突破点,在回调结束时顺势入场。

**Al Brooks术语**: Breakout Pullback, Failed Pullback  
**太妃原始术语**: 区间突破回调

---

## 📋 入场条件 (Entry Criteria)

### 市场背景要求
- [x] 市场周期: **区间突破** 或 **新趋势形成**
- [x] 波段结构: 区间突破已确认
- [x] 动能特征: 突破后首次回调

### 关键识别要素
1. **区间突破确认**:
   * 价格强势突破区间边界
   * 突破K线收盘远离区间(非假突破)
   * 突破后价格持续远离区间
2. **回调特征**:
   * 首次回调回到突破点附近
   * 回调深度通常25%-50%
   * 回调测试突破点不破(Failed Pullback)
3. **入场信号**:
   * 回调结束,价格重新向突破方向运动
   * 出现High 1/Low 1突破信号
   * 理想情况:回调止步于突破点,形成二次入场

### 信号棒特征
- [x] 位置: 突破点附近(区间边界)
- [x] K线形态: High 1/Low 1或反转棒
- [x] 确认: 价格重新向突破方向突破
- [x] 理想: 回调浅且快速恢复

### 特殊注意
* **与看衰突破的区别**: 
  * 看衰突破:区间内,预期突破失败
  * 突破回调:突破后,预期回调失败(趋势延续)
* **突破真假判断**: 突破K线必须收盘远离区间
* **回调深度**: 回调超过50%可能转为新区间

---

## 🛡️ 风险管理 (Risk Management)

### 止损位置
* **标准**: 突破点外2-4跳
  * 向上突破:止损在区间上缘下方
  * 向下突破:止损在区间下缘上方
* **保守**: 回调极值外
  * 如果回调形成明确低点,止损设在低点外

### 目标位计算
* **首选**: 区间高度的等距运动(AB=CD)
  * A=区间底, B=区间顶, C=突破点
  * D=C + (B-A) = 目标位
* **次选**: 前方磁体位置
  * 老高/老低
  * 整数关口
  * 缺口位置
* **最小目标**: 确保盈亏比≥2:1

### 入场时机选择
* **激进**: 回调触及突破点立即入场
  * 优点:最佳盈亏比
  * 风险:可能继续回调
* **标准**: 等待High 1/Low 1确认
  * 回调结束后价格重新突破
* **保守**: 等待第二次回调不破(Second Entry)
  * 最高确认度,但可能错过

## ⚠️ 失败模式 (Failure Signals)

当出现以下情况,立即退出或不入场:
- [x] **突破失败**: 价格重新回到区间内部
- [x] **回调过深**: 回调超过突破段的50%-75%
- [x] **假突破**: 突破K线未能收盘远离区间
- [x] **转为新区间**: 在突破点附近横盘整理
- [x] **动能衰竭**: 第二推动能明显减弱

### 真突破vs假突破
| 真突破(可交易) | 假突破(避免) |
|-------------|------------|
| 收盘远离区间 | 收盘在区间附近 |
| 持续推进 | 立即回落 |
| 动能强劲 | 动能不足 |
| 首次回调浅 | 首次回调深 |
| 回调测试不破 | 回调突破失败 |

## 📊 Al Brooks理论对应

### 核心概念映射
| 太妃术语 | Al Brooks术语 | 说明 |
|---------|--------------|------|
| 区间突破回调 | Breakout Pullback | 突破后回调 |
| 测试突破点 | Test of Breakout Point | 回测区间边界 |
| 回调失败 | Failed Pullback | 回调无法破突破点 |
| 等距运动 | Measured Move | 目标位计算 |
| 二次入场 | Second Entry | 第二次入场机会 |

### 适用的Al Brooks形态
- **Breakout Pullback**: 核心形态
- **High 1/Low 1**: 回调后的突破信号
- **Measured Move**: 目标位计算(区间高度)
- **Failed Pullback**: 回调失败,趋势延续

## 🎓 学习笔记 (Notes)

### 课程来源
* **视频**: 太妃价格行为 L15B - 急赴磁体-区间突破回调
* **时间码**: 后半段 - 区间突破回调

### 关键要点
1. **区间突破的两种机会**:
   * 突破时:直接跟随(风险较大)
   * 回调时:等待回调后入场(风险较小)
2. **回调深度**: 通常25%-50%,超过75%可能失败
3. **测试不破**: 回调测试突破点但不破是最佳信号
4. **等距运动**: 区间高度提供目标位测算
5. **与逆1顺1的关系**: 本质都是首次回调后的顺势入场

### 个人理解
* 区间突破回调是最经典的趋势跟随策略
* 比直接追突破更安全,盈亏比更好
* 关键是判断"真突破"还是"假突破"
* 回调深度是重要指标:浅回调说明趋势强
* Al Brooks整本书都在讲这个形态的各种变体

## 🔄 操作检查清单 (Checklist)

**入场前检查**:
- [ ] 识别区间:明确的上下边界
- [ ] 确认突破:收盘远离区间,持续推进
- [ ] 观察回调:首次回调已开始
- [ ] 评估深度:回调是否接近突破点
- [ ] 等待信号:High 1/Low 1或测试不破

**持仓中监控**:
- [ ] 观察第二推:向目标位(等距运动)推进
- [ ] 警惕破突破点:如果回调破突破点则止损
- [ ] 关注磁体:前方磁体可能减速

**复盘问题**:
- [ ] 突破是真突破还是假突破?
- [ ] 回调深度是否合理(≤50%)?
- [ ] 测试突破点时是否坚守不破?
- [ ] 目标位计算是否合理?
- [ ] 如果失败,是否回调过深或动能不足?

> [!tip] 实战建议
> 1. **新手**: 等待明确的High 1/Low 1确认,不要过早入场
> 2. **中级**: 可在突破点附近挂限价单,等待回调触发
> 3. **高手**: 区分真假突破,在真突破的回调时果断入场

> [!warning] 风险提示
> * 必须等突破确认(收盘远离区间)才考虑
> * 回调超过50%说明突破可能失败
> * 假突破会导致被套,严格止损
> * 前方强磁体可能阻挡等距运动
> * 不要在突破当下追单,等回调
> * 如果价格重回区间内,立即止损
