# 🎯 策略概览

**一句话描述**: 
价格两次测试同一支撑/阻力位失败后,在第二次测试处反向入场,捕捉失败突破的反转机会。

**Al Brooks术语**: Double Top/Bottom, Failed Breakout  
**太妃原始术语**: 双重顶底、C型顶底

---

## 📋 入场条件 (Entry Criteria)

### 市场背景要求
- [x] 市场周期: **交易区间** 或 **弱趋势/通道**
- [x] 波段结构: 至少两次测试同一极值
- [x] 动能特征: 第二次测试动能减弱

### 关键识别要素
1. **双重定义**:
   * 价格两次测试同一支撑/阻力位
   * 第二次测试失败(无法突破或立即反向)
   * 与楔形区别: 双重只有二推,楔形有三推
2. **测试类型**:
   * **持平**: 第二次测试与第一次几乎相同(±2-3跳)
   * **过冲**: 第二次略微突破第一次极值
   * **不及**: 第二次未能到达第一次极值
3. **操作优先级**: 持平或过冲的形态更易操作,胜率更高

### 信号棒特征
- [x] 位置: 第二次测试处
- [x] K线形态: 反转棒(强收盘在相反方向)
- [x] 确认: 突破双顶/双底之间的颈线
- [x] 理想: 第二次测试后立即出现强劲反向K线

### 特殊注意
* **区间内的双重**: 更可靠,属于区间交易
* **趋势中的双重**: 可能演变为旗形继续原趋势
* **低推回调的双重**: 在上升趋势中,低推回调被二次突破失败是做空信号

---

## 🛡️ 风险管理 (Risk Management)

### 止损位置
* **标准**: 第二次测试极值外2-4跳
  * 双顶: 第二个高点上方
  * 双底: 第二个低点下方
* **保守**: 第一次测试极值外
  * 适用于资金量较大或风险厌恶型交易者

### 目标位计算
* **首要目标**: 颈线对侧的等距投射
  * 双顶: 从颈线向下投射双顶高度
  * 双底: 从颈线向上投射双底高度
* **次要目标**: 区间对侧(如在区间内)
* **最小目标**: 确保盈亏比≥1.5:1

### 入场时机选择
* **激进**: 第二次测试处直接反向入场
  * 风险: 可能出现第三次测试(演变为楔形)
* **保守**: 等待突破颈线后入场
  * 优点: 确认反转
  * 缺点: 盈亏比略差
* **最保守**: 等待重测颈线不破后入场(Second Entry)
  * 最高确认度,但可能错过机会

---

## ⚠️ 失败模式 (Failure Signals)

当出现以下情况,立即退出或不入场:
- [x] **第三次测试**: 演变为楔形(三推),需重新评估
- [x] **突破成功**: 第二次测试成功突破,转为趋势延续
- [x] **横盘整理**: 双顶/双底之间形成紧密区间
- [x] **动能增强**: 第二次测试比第一次更强劲
- [x] **趋势中的双重**: 在强趋势中途,可能只是回调

### 双重vs楔形的判断
| 特征 | 双重顶底 | 楔形顶底 |
|------|---------|---------|
| 推动次数 | 二推 | 三推 |
| 收敛要求 | 无要求 | 倾向收敛 |
| 反转概率 | 中等 | 较高 |
| 失败后果 | 可能演变为楔形 | 可能演变为旗形 |

## 📊 Al Brooks理论对应

### 核心概念映射
| 太妃术语 | Al Brooks术语 | 说明 |
|---------|--------------|------|
| 双重顶底 | Double Top/Bottom | 两次测试失败 |
| C型顶底 | Failed Breakout | 突破失败形态 |
| 颈线 | Neckline | 双顶/双底之间的支撑/阻力 |
| 持平/过冲/不及 | Equal/Overshoot/Undershoot | 第二次测试的三种类型 |

### 适用的Al Brooks形态
- **Major Trend Reversal (MTR)**: 大级别双顶/双底
- **Failed Breakout**: 突破失败
- **Trading Range Reversal**: 区间内的反转
- **Second Entry**: 第二次入场机会(重测颈线)

## 🎓 学习笔记 (Notes)

### 课程来源
* **视频**: 太妃价格行为 L14B - 双重顶底-楔形
* **时间码**: 02:58 - 双重顶底交易

### 关键要点
1. **优先级**: 推荐交易持平或过冲的形态,更易操作
2. **位置重要**: 区间内的双重比趋势中的更可靠
3. **与楔形关系**: 双重失败可能演变为楔形(第三次测试出现)
4. **低推回调**: 在上升趋势中是特殊的做空信号

### 个人理解
* 双重顶底是最经典的反转形态,但可靠性中等
* 关键是"两次",如果出现第三次就要重新判断(楔形)
* 区间内的双重胜率更高,因为有明确的支撑/阻力
* Al Brooks的MTR形态本质就是大级别的双重顶底

## 🔄 操作检查清单 (Checklist)

**入场前检查**:
- [ ] 识别第一次测试位置(第一个高点/低点)
- [ ] 确认第二次测试: 持平/过冲/不及?
- [ ] 观察第二次测试后的反向K线
- [ ] 评估位置: 区间内还是趋势中?
- [ ] 计算盈亏比≥1.5:1

**持仓中监控**:
- [ ] 警惕第三次测试出现(演变为楔形)
- [ ] 观察是否向目标位运动
- [ ] 关注颈线突破确认

**复盘问题**:
- [ ] 双重识别是否正确?(不是楔形)
- [ ] 第二次测试类型判断准确吗?
- [ ] 入场时机选择合理吗?
- [ ] 止损/目标位设置是否合理?

> [!tip] 实战建议
> 1. **新手**: 只交易区间内的持平双重,等待颈线突破确认
> 2. **中级**: 可交易过冲双重,在第二次测试处直接入场
> 3. **高手**: 区分区间双重(高胜率)vs趋势双重(低胜率)

> [!warning] 风险提示
> * 趋势中途的双重可能只是回调,反转概率较低
> * 警惕演变为楔形(第三次测试出现)
> * 不及型双重的可靠性低于持平和过冲型
> * 必须有明确的反向K线确认,不可盲目入场
> * 区间外的双重需要更大止损,盈亏比可能不足
