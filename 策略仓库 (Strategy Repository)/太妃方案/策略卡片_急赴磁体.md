# 🎯 策略概览

**一句话描述**: 
当价格接近区间边线且出现连续趋势K线时,预期价格将快速触及磁体(区间边线),顺势入场捕捉最后一段。

**Al Brooks术语**: Magnet Trade / Rush to Support/Resistance  
**太妃原始术语**: 急赴磁体、趋势K线交易

---

## 📋 入场条件 (Entry Criteria)

### 市场背景要求
- [x] 市场周期: **交易区间** 或 **突破模式**
- [x] 磁体识别: 明确的区间边线(支撑/阻力)
- [x] 距离要求: 价格距离磁体足够近(≤1/4区间高度)

### 关键识别要素
1. **磁体定义**:
   * 区间上缘/下缘
   * 前期显著高点/低点
   * 整数关口(如5000, 10000等)
   * 未补缺口
2. **距离计算**:
   * 丈量区间高度(从上缘到下缘)
   * 确认收线位置距离边线≤25%区间高度
   * 参数可调整为20%-33%,不建议超过1/3
3. **动能要求**:
   * 前赴磁体的动能足够流畅
   * 趋势K线接踵而来
   * 运动斜率、连续性表现良好

### 信号棒特征
- [x] K线形态: **接连的趋势K线**(大阳/大阴)
- [x] 方向: 朝向立极方向(向磁体方向)
- [x] 动能: 无明显重叠,推进流畅
- [x] 收盘: 强收盘,接近K线极值

### 路径分析
- [ ] 确认前方磁体位置明确
- [ ] 评估距离是否≤1/4区间高度
- [ ] 观察趋势K线连续性

## 🛡️ 风险管理 (Risk Management)

### 止损位置
* **标准**: 前一个波段点之外
  * 牛市: 回调低点下方2-4跳
  * 熊市: 回调高点上方2-4跳
* **替代**: 区间边线之外
  * 牛市急赴上缘: 区间下缘下方
  * 熊市急赴下缘: 区间上缘上方

### 目标位计算
* **首要目标**: 磁体位置(区间边线)
* **盈亏比特点**: 
  * 距离越近,盈亏比越差但胜率越高
  * 通常1:1至1.5:1
  * 通过高胜率补偿盈亏比不足

### 入场方式
* **激进**: 市价单入场(当实体收在关键线之下/之上时)
  * 优点: 不会错过机会
  * 缺点: 如动能不足可能立即止损
* **保守**: 停止订单入场(在针尖被突破后)
  * 优点: 确认动能后入场
  * 缺点: 可能错过部分利润

### 仓位管理
* **标准仓位**: 由于盈亏比较低,不建议重仓
* **快进快出**: 到达磁体后立即获利,不贪婪

---

## ⚠️ 失败模式 (Failure Signals)

当出现以下情况,立即退出或不入场:
- [x] **距离过远**: 距离磁体>1/3区间高度
- [x] **动能不足**: K线开始重叠,出现内包线
- [x] **横盘整理**: 价格在途中形成横向区间
- [x] **反向突破**: 突然出现强劲反向K线
- [x] **磁体已触及**: 针尖刚触及后出现内包K线(可能已是反转)

### 磁体触及后的处理
* **保守策略**: 磁体刚被触及后出现内包K线时不介入
  * 原因: 针尖触及可能已是反转开始
* **激进策略**: 价格转高至中轴附近后再度回测磁体时
  * 设置停止订单于折返低点被重新突破时

## 📊 Al Brooks理论对应

### 核心概念映射
| 太妃术语 | Al Brooks术语 | 说明 |
|---------|--------------|------|
| 急赴磁体 | Rush to Magnet | 快速接近支撑/阻力 |
| 磁体 | Magnet | 吸引价格的关键位 |
| 趋势K线 | Trend Bar | 大阳线/大阴线 |
| 区间边线 | Trading Range Boundary | 支撑/阻力位 |
| 立极方向 | Direction of Extreme | 朝向高/低点方向 |

### 适用的Al Brooks形态
- **Trading Range Magnet**: 区间内的磁体效应
- **Final Push to Support/Resistance**: 最后冲刺至关键位
- **Scalp in Trading Range**: 区间内的剥头皮

### 课程来源

* **时间码**: 02:03 - 交易流程
* **时间码**: 04:49 - 理想要件

### 关键要点
1. **距离要求核心**: 25%的参数可调整,但不建议超过1/3
2. **盈亏比权衡**: 距离越近胜率越高但盈亏比越差
3. **动能评估关键**: 必须观察趋势K线的样态和连续性
4. **新手建议**: 采用25%的标准参数,避免过于激进
5. **磁体触及后**: 保守不介入,激进等二次回测

### 个人理解
* 这是典型的"剥头皮"策略,目标小但胜率高
* 关键是"急",如果动能不流畅就跳过
* 1/4区间高度是经验值,类似技术分析中的"最后1/3"
* 适合日内交易者,不适合波段交易

## 🔄 操作检查清单 (Checklist)

**入场前检查**:
- [ ] 识别磁体: 区间边线位置明确
- [ ] 丈量距离: 计算当前价格到磁体的距离
- [ ] 距离确认: ≤25%区间高度(最多不超过33%)
- [ ] 动能评估: 趋势K线连续,无明显重叠
- [ ] 入场方式: 市价还是停止订单?

**持仓中监控**:
- [ ] 观察是否继续向磁体推进
- [ ] 如出现内包/重叠考虑离场
- [ ] 接近磁体时准备获利

**复盘问题**:
- [ ] 距离计算是否准确?
- [ ] 动能评估是否正确?
- [ ] 入场时机是否最佳?
- [ ] 是否到达磁体后立即获利?

> [!tip] 实战建议
> 1. **新手**: 使用25%距离标准,停止订单入场,到达磁体立即获利
> 2. **中级**: 可尝试20%距离,提高盈亏比,但需更严格筛选动能
> 3. **高手**: 可在磁体触及后等待二次回测,进行反向交易

> [!warning] 风险提示
> * 盈亏比较低(1:1至1.5:1),依赖高胜率,不可频繁使用
> * 动能不足时强行入场容易止损
> * 距离过远(>1/3)时胜率大幅下降
> * 到达磁体后不要贪婪,立即获利离场
> * 该策略更适合日内交易,不适合隔夜持仓
