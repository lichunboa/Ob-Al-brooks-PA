# 🎯 策略概览

**一句话描述**: 
在强趋势中,首次逆势尝试(逆1)失败后,等待顺势恢复(顺1)的高1或低1入场点。

**Al Brooks术语**: High 1 / Low 1, First Counter-trend Failure  
**太妃原始术语**: 逆1失败/顺1成功

---

## 📋 入场条件 (Entry Criteria)

### 市场背景要求
- [x] 市场周期: **强趋势** 或 **急速突破**
- [x] 波段结构: 突破已确认,价格远离突破点
- [x] 动能特征: 趋势初期动能强劲

### 关键识别要素
1. **逆1失败定义**:
   * 上升趋势中: 空头首次反转尝试(Low 1)失败
   * 下降趋势中: 多头首次反转尝试(High 1)失败
   * 关键: "首次"逆势尝试,不是第二次或第三次
2. **顺1成功定义**:
   * 上升趋势: 多头恢复趋势的High 1成功
   * 下降趋势: 空头恢复趋势的Low 1成功
3. **核心原理**: 
   * 趋势具有延续性
   * 首次逆势尝试通常失败(除非遇到强磁体)
   * 顺势恢复后有第二推等距运动

### 信号棒特征
- [x] 位置: 逆1失败后,价格回归趋势方向
- [x] K线形态: High 1 或 Low 1 突破
- [x] 确认: 突破逆1的极值(上升趋势突破Low 1高点)
- [x] 理想: 逆1回调幅度浅,顺1动能强

### 特殊注意
* **突破标准**: 价格收于重要阻力位外 + 持续远离突破点
* **两种建仓方式**:
  * 市价追进: 在连续强K线上直接入场(实体投影测算)
  * 逆1顺1: 等待回调后High 1/Low 1入场(AB=CD测算)
* **动能甄选**: "突破要远,测试要浅" - 突破距离越远动能越强

## 🛡️ 风险管理 (Risk Management)

### 止损位置
* **标准**: 逆1的极值外2-4跳
  * 上升趋势: Low 1低点下方
  * 下降趋势: High 1高点上方
* **保守**: 急速起点或突破点外

### 目标位计算
* **首选**: AB=CD等距运动
  * A=突破点, B=逆1极值, C=顺1入场点
  * D=目标位 (C + (B-A))
* **次选**: 实体投影测算(市价追进时)
  * 基于动能交易者逻辑
  * 采用强K线实体幅度投影
* **磁体调整**: 结合前方磁体(老高/老低)调整目标

### 入场时机选择
* **激进**: 在确认K线高点挂停止单,突破即入场
  * 优点: 捕捉最佳位置
  * 风险: 可能假突破
* **保守**: 等待突破后回测不破再入场(Second Entry)
  * 优点: 更高确认度
  * 缺点: 盈亏比略差

## ⚠️ 失败模式 (Failure Signals)

当出现以下情况,立即退出或不入场:
- [x] **逆1成功**: 首次逆势尝试成功突破,转为趋势反转
- [x] **动能衰退**: 突破距离不足,后续幅度递减
- [x] **测试过深**: 回调幅度超过突破段的50%
- [x] **磁体阻力**: 前方遇到强磁体(老高/老低,缺口等)
- [x] **第二次逆势**: 已经出现过逆1顺1,现在是第二次逆势尝试

### 动能衰退识别
| 健康趋势 | 衰退趋势 |
|---------|---------|
| 突破远离突破点 | 突破距离递减 |
| 回调浅(≤50%) | 回调深(>50%) |
| 顺1动能强 | 顺1动能弱 |
| 连续新高/新低 | 极值不再递进 |

## 📊 Al Brooks理论对应

### 核心概念映射
| 太妃术语 | Al Brooks术语 | 说明 |
|---------|--------------|------|
| 逆1失败 | First Counter-trend Failure | 首次逆势尝试失败 |
| 顺1成功 | High 1 / Low 1 | 顺势恢复的第一次突破 |
| 等距运动 | Measured Move (AB=CD) | 目标位测算 |
| 突破要远 | Strong Breakout | 动能甄选标准 |
| 测试要浅 | Shallow Pullback | 回调深度要求 |

### 适用的Al Brooks形态
- **High 1 / Low 1**: 核心形态
- **Spike and Channel**: 急速后进入通道,逆1顺1常出现
- **Measured Move**: 目标位计算方式
- **Second Entry**: 保守入场时机

## 🎓 学习笔记 (Notes)

### 课程来源
* **视频**: 太妃价格行为 L13B - 急速交易-逆1顺1
* **时间码**: 00:03 - 急速建仓方案之二

### 关键要点
1. **两种建仓方式的差异**:
   * 市价追进: 无回调,实体投影,动能交易者逻辑
   * 逆1顺1: 有回调,AB=CD测算,价值交易者逻辑
2. **动能甄选**: "突破要远,测试要浅"是核心标准
3. **连续使用**: 两种方案可在同一趋势中连续应用
4. **趋势初期最佳**: 越早入场数学优势越大

### 个人理解
* 逆1顺1本质是Al Brooks的High 1/Low 1形态
* 关键是"首次"逆势尝试,第二次或第三次可靠性降低
* 与20均线缺口的区别: 逆1顺1不依赖均线,纯粹看K线结构
* 在Spike后的Channel阶段最常见

## 🔄 操作检查清单 (Checklist)

**入场前检查**:
- [ ] 确认趋势: 突破已确认且价格远离突破点
- [ ] 识别逆1: 首次逆势尝试已出现
- [ ] 判断失败: 逆1未能突破,开始回归趋势
- [ ] 评估动能: 突破远测试浅(健康趋势)
- [ ] 设置订单: 在确认K线高/低点挂停止单

**持仓中监控**:
- [ ] 观察目标: 向AB=CD目标位运动
- [ ] 警惕逆1成功: 如果逆1突破则止损
- [ ] 关注磁体: 前方磁体可能减速或反转

**复盘问题**:
- [ ] 是否真的是"首次"逆势尝试?
- [ ] 突破动能是否充足(远离突破点)?
- [ ] 回调深度是否合理(≤50%)?
- [ ] 目标位计算方式选择正确吗?
- [ ] 如果失败,是否有动能衰退迹象?

> [!tip] 实战建议
> 1. **新手**: 只在强趋势初期交易,等待High 1/Low 1明确突破
> 2. **中级**: 可在确认K线高点挂停止单,提前捕捉突破
> 3. **高手**: 区分急速段(实体投影)vs通道段(AB=CD)

> [!warning] 风险提示
> * 必须是"首次"逆势尝试,第二次可靠性大幅降低
> * 突破距离不足说明动能弱,不适合交易
> * 回调超过50%说明趋势可能结束
> * 前方强磁体可能导致等距运动失败
> * 避免在动能衰退时强行交易
