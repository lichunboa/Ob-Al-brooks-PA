# 🎯 策略概览

**一句话描述**: 
价格首次穿透20EMA对侧形成缺口后又返回原趋势侧,在均线附近入场捕捉测试原极值的机会。

**Al Brooks术语**: First Moving Average Gap / First Pullback After MA Cross  
**太妃原始术语**: 第一均线缺口、首次回返均线缺口

---

## 📋 入场条件 (Entry Criteria)

### 市场背景要求
- [x] 市场周期: **弱趋势/通道**(动能减弱阶段)
- [x] 波段结构: 从强趋势转为通道
- [x] 均线关系: 价格曾穿透均线对侧,现返回原趋势侧

### 关键识别要素
1. **形成过程**:
   * Step 1: 价格首次穿透20EMA到对侧(牛市穿到下方/熊市穿到上方)
   * Step 2: 价格又返回到原趋势侧
   * Step 3: 在均线附近形成缺口区域(灰紫色区块)
2. **K线数量**: 
   * 独立出现: 20-30根标准
   * 接续20均线缺口: 可放宽至30-45根
3. **动能特征**: 容忍均线短暂穿透,反映趋势动能减弱

### 信号棒特征 - H1/L1形态
- [x] K线形态: **同侧高低1**(High 1 牛市 / Low 1 熊市)
- [x] 收盘位置: 强收盘
- [x] 位置要求: 在20EMA附近或略微穿透
- [x] 动能筛选: 避免强劲趋势K线连续穿透均线

### 风险评估
- ⚠️ **胜率低于20均线缺口**: 因趋势动能减弱
- ✅ **更高盈亏比**: 补偿较低胜率
- ⚠️ **双重风险**: 若与20均线缺口同时持仓需防双重止损

---

## 🛡️ 风险管理 (Risk Management)

### 止损位置
* **推荐**: 信号K线外(a2方案)
  * 牛市: Entry Bar低点下方1-2跳
  * 熊市: Entry Bar高点上方1-2跳
  * 原因: 该策略胜率较低,需提高盈亏比
* **保守**: 前波段点外(a1方案)
  * 风险: 盈亏比可能不足2:1

### 目标位计算
* **首要目标**: 原趋势极值(Old High/Low)
* **延伸目标**: 突破极值后的等距投射
* **最小要求**: 确保盈亏比≥2.5:1

### 仓位管理
* **谨慎建仓**: 初始仓位应小于20均线缺口策略
* **避免叠加**: 如已有20均线缺口仓位,需评估双重风险
* **优先级**: 20均线缺口优先级更高

## ⚠️ 失败模式 (Failure Signals)

当出现以下情况,立即退出或不入场:
- [x] **连续穿透**: 强劲趋势K线连续穿透均线,动能未减弱
- [x] **深度调整**: 价格在均线对侧停留过久
- [x] **横盘整理**: 在均线附近形成交易区间
- [x] **二次穿透**: 价格再次穿透到均线对侧
- [x] **H1失败**: 同侧高低1信号被立即突破

### 与20均线缺口的区别
| 特征 | 20均线缺口 | 第一均线缺口 |
|------|-----------|-------------|
| 均线关系 | 从未穿透对侧 | 曾穿透对侧又返回 |
| 趋势强度 | 强趋势(Spike) | 弱趋势(Channel) |
| 胜率 | 较高(~65%) | 较低(~55%) |
| 盈亏比 | 2:1 | 2.5:1以上 |
| K线数量 | 严格20-30 | 可放宽30-45 |

## 📊 Al Brooks理论对应

### 核心概念映射
| 太妃术语 | Al Brooks术语 | 说明 |
|---------|--------------|------|
| 第一均线缺口 | First MA Gap / First Reentry | 首次穿透后返回 |
| 均线对侧 | Opposite Side of EMA | 牛市的下方/熊市的上方 |
| 首次回调序列 | First Pullback Sequence | 倾向失败的反转 |
| 动能减弱 | Weakening Momentum | 从Spike转Channel |

### 适用的Al Brooks形态
- **Transition from Spike to Channel**: 急速转通道阶段
- **First Pullback in Weakening Trend**: 趋势减弱中的首次回调
- **Test of EMA after Overshoot**: 过冲后的均线测试

## 🎓 学习笔记 (Notes)

### 课程来源
* **视频**: 太妃价格行为 L17B - 20均线缺口-第一均线缺口
* **时间码**: 01:19 - 第一均线缺口定义
* **时间码**: 09:06 - 操作细则

### 关键要点
1. **命名争议**: 太妃认为命名不统一,建议改为"首次回返均线缺口"
2. **典型序列**: 通常先出现20均线缺口,随后出现第一均线缺口
3. **核心规律**: 首次回调倾向成为失败的反转,最终测试原极值
4. **特殊设置**: 止损多采用信号K线外,提高盈亏比

### 个人理解
* 这是趋势"第二阶段"的策略,动能已不如20均线缺口
* 关键识别点: 价格曾经"过河"(穿透均线)又"回来"
* 胜率较低但盈亏比高,适合风险偏好较高的交易者
* 需要更严格的信号筛选,避免低质量入场

## 🔄 操作检查清单 (Checklist)

**入场前检查**:
- [ ] 确认价格曾穿透均线对侧(看历史K线)
- [ ] 价格已返回原趋势侧
- [ ] 识别H1/L1信号棒
- [ ] 动能评估: 避免强劲连续穿透
- [ ] 计算盈亏比≥2.5:1

**持仓中监控**:
- [ ] 警惕价格再次穿透均线对侧
- [ ] 观察是否向原极值运动
- [ ] 如出现横盘考虑提前离场

**复盘问题**:
- [ ] 是否正确识别了"穿透对侧"?
- [ ] H1/L1信号质量如何?
- [ ] 止损设置是否合理(应为a2方案)?
- [ ] 与20均线缺口混淆了吗?
