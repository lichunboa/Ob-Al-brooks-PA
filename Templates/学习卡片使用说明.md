# 📚 学习卡片使用说明

## 卡片语法

### 1. 单行卡片（问答式）
```
问题 :: 答案
```
- 前面是问题，后面是答案
- 示例：`什么是支撑位 :: 价格下跌时难以突破的水平线`

### 2. 双向卡片
```
答案 ::: 问题
```
- 问题和答案可以互换测试
- 统计为两张卡片

### 3. 多行卡片
```
正面问题（可多行）
?
背面答案（可多行）
```

### 4. 多行双向卡片
```
正面（可多行）
??
背面（可多行）
```
- 统计为两张卡片

### 5. 填空卡片（支持多种格式）

| 格式 | 示例 | 说明 |
|------|------|------|
| **高亮** | `这是==答案==的问题` | Obsidian 原生高亮 |
| **双大括号** | `这是{{答案}}的问题` | SRS 官方格式 |
| **Anki 语法** | `这是{{c1::答案}}的问题` | 兼容 Anki 导入 |

> ⚠️ **注意**：模板变量如 `{{date}}`、`{{time}}` 不会被识别为填空题

## 复习标记格式

每张卡片复习后会自动添加标记：
```
<!--SR:!2025-12-22,4,270-->
```

| 字段           | 含义                |
| ------------ | ----------------- |
| `2025-12-22` | 下次复习日期            |
| `4`          | 当前间隔天数            |
| `270`        | 难度分数（Ease Factor） |

### 难度分数说明
- **初始值**：250（默认）
- **范围**：通常在 130-310 之间
- **意义**：分数越高，卡片越"简单"，间隔增长越快

## 🧠 SRS 复习推荐算法

### 卡片优先级排序

1. **到期卡片优先**：先显示所有已到期（`due_date <= today`）的卡片
2. **按到期日期排序**：最早到期的卡片先复习
3. **新卡片**：所有到期卡片复习完后，才会显示新卡片
4. **PageRank 排序**：新卡片按笔记链接数量排序（链接越多越重要）

### 复习按钮对应的算法

| 按钮 | 间隔变化 | 难度变化 | 适用场景 |
|------|----------|----------|----------|
| **简单** | 间隔 × 2.5+ | 难度 +20 | 轻松回忆，无需犹豫 |
| **记得** | 间隔 × 2.0 | 难度不变 | 正常回忆成功 |
| **较难** | 间隔 × 1.5- | 难度 -20 | 勉强记得，有些困难 |
| **重来** | 间隔重置为1天 | 难度显著下降 | 完全忘记 |

### 间隔计算公式

```
新间隔 = 当前间隔 × (难度分数 / 100) × 按钮系数
```

- **简单系数**：约 1.3
- **记得系数**：约 1.0
- **较难系数**：约 0.8

## 🎮 控制台集成功能

### 教练焦点 (Coach Focus)

控制台的学习模块提供以下 SRS 集成功能：

| 功能 | 说明 |
|------|------|
| **SRS 记忆曲线推荐** | 显示最紧急需要复习的笔记 |
| **📖 复习** | 正常复习（只复习到期卡片） |
| **🔥 强化** | 强化复习（忽略间隔，复习所有卡片） |
| **随机抽题** | 从题库随机抽取卡片测试 |
| **🔗 SRS** | 跳转到 SRS 复习当前笔记 |
| **✏️ 编辑** | 跳转到笔记具体行编辑卡片 |
| **🔀 换一批** | 重新随机抽取一批新题目 |

### 随机抽题支持

- ✅ **Markdown 渲染**：问题和答案支持图片、链接、格式
- ✅ **行号跳转**：点击编辑按钮直接跳转到卡片所在行
- ✅ **策略关联**：显示卡片关联的策略和胜率

## 📊 示例解读

### 示例 1：简单
```
什么是趋势线 :: 连接高点或低点的直线
<!--SR:!2025-12-22,4,270-->
```
- 下次复习：2025-12-22
- 当前间隔：4天
- 难度分数：270（较高，说明掌握较好）

### 示例 2：较难
```
==交易区间==不适合交易的原因是什么？
震荡行情难以盈利
<!--SR:!2025-12-21,2,232-->
```
- 下次复习：2025-12-21
- 当前间隔：2天（较短）
- 难度分数：232（较低，需要多复习）

### 示例 3：双向卡片
```
缺口 ::: 要交易
<!--SR:!2025-12-22,3,250!2000-01-01,1,250-->
```
- 第一个标记：正向测试（问→答）
- 第二个标记：反向测试（答→问）

## 💡 学习建议

1. **诚实评估**：根据实际回忆难度选择按钮，不要高估自己
2. **及时复习**：尽量在提示日期完成复习，保持记忆曲线
3. **控制新卡**：每天新学卡片不宜过多（建议 10-20 张）
4. **关注难点**：难度分数低于 200 的卡片需要重点关注
5. **使用随机抽题**：利用控制台的随机抽题功能进行自测

## 🔧 可用命令

在 Obsidian 命令面板（`Ctrl/Cmd + P`）中可使用：

| 命令 | 功能 |
|------|------|
| `Spaced Repetition: Review flashcards` | 开始复习所有卡片 |
| `Spaced Repetition: Cram flashcards` | 填鸭式复习（忽略间隔） |
| `Spaced Repetition: Review flashcards in note` | 只复习当前笔记的卡片 |
| `Spaced Repetition: Open review queue view` | 打开复习队列视图 |
