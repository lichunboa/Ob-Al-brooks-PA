# context的重要性例子

- 同样的BOM在不同context下的例子会有不同的价格共识
    - case 1:
        
        之前趋势：bear
        
        BOM的位置：EMA下方 bear BO
        
        价格共识：这里的BOM理解为bear trend的回调，然后继续bear趋势。
        
    - case 2:
        
        之前趋势: bull 
        
        BOM的位置：EMA上方
        
        价格共识：这里的BOM更容易理解为bull trend的回调，而不是直接向下。
