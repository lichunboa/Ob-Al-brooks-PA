# 📺 1. 课程概览 (Module Overview)

**本节核心**：急速=突破动能阶段；“收线追进 (BUYNOW)”的前提是突破有跟进、远离磁体、路径无阻力，并且止损/目标必须有价格行为与数学优势支撑。

**核心知识点**
1. 方案必须同时满足：价格行为理据 + 数学优势
2. 急速结束条件：出现首根回调K线且低点开始降低（趋势动能衰退）
3. 窄幅通道 vs 宽幅通道：逆势方是否能用限价/止损盈利
4. 收线追进：止损放在推动起点外，目标用等距投射（AB=CD/实体投影）

#flashcards

一个可盈利的建仓方案必须同时满足什么两点？ :: 价格行为理据 + 数学优势（统计有效性）。
急速（spike）的定义要点？ :: 连续同向K线推进、基本不允许回调（严格版本），本质属于突破动能阶段。
急速结束的标志是什么？ :: 出现首根回调K线且低点开始降低（动能阶段结束，进入通道/调整）。
窄幅通道 vs 宽幅通道的根本区别？ :: 逆势方能否通过限价/止损订单盈利；窄幅通道逆势方依然难以获利，动能更强。
BUYNOW 收线追进的适用条件（至少三条）？ :: 突破有强跟进；突破点远离显著磁体；路径无明显阻力障碍。
BUYNOW 的止损应放哪里？ :: 推动起点之外（基于重要波段点的价格行为理由），不是“半路随意收紧”。
BUYNOW 的目标怎么定？ :: 等距投射（AB=CD/实体高度投影），优先与关键位/磁体重合。
==BUYNOW== 追进的三前提：==跟进==、==远离磁体==、==路径无阻力==。 :: （填空）
BUYNOW 的本质是什么？ :: 在强突破动能阶段收线市价追进，赌“还有跟进 + 等距延续”。
情景：突破后立刻出现大上影线且附近有磁体，你对 BUYNOW 的态度？ :: 降低/避免追进；更倾向等待回调或把它视为高潮后衰竭风险。
BUYNOW 最常见的失败模式是什么？ :: 高潮后衰竭/假突破，随后深回调触发宽止损。
为什么 BUYNOW 的止损要放在“推动起点外”？ :: 才能避开正常回调噪音、给趋势留空间；否则会被正常回调扫掉并破坏数学优势。
目标定等距（AB=CD/实体投影）的逻辑是什么？ :: 动能阶段常出现等距推进，用客观测量约束主观情绪与贪恐。
窄幅通道里为什么逆势方难赚钱？ :: 顺势动能强、停顿少，逆势限价/止损订单缺乏统计优势。
如何识别动能衰退进入通道/调整？ :: 第一根回调K出现且低点开始降低（或连续停顿、跟进变差）。

### 1）急速 05:16

### 3）动能交易者的数学优势

- 统计优势：趋势中80%以上的反转尝试会失败，源于动能交易者的集体行为形成的市场惯性。
