# 🧠 1. 概览

- 急速（Climax）与窄幅通道阶段，是“动能交易者占优”的趋势延续区。
- 可盈利的建仓方案必须同时具备：价格行为理据（解释机制）+ 数学优势（统计正期望）。
- “收线市价追进（BUYNOW）”适用于突破后连续强势趋势 K 线：每根 K 线收线追进，止损放在推动起点之外，目标用等距/AB=CD。
- 止损必须基于价格行为（重要波段点），而不是基于“我能承受多少”。

# ⚡ 2. 速查表

| 主题            | 要点                                                    | 典型错误                  |
| --------------- | ------------------------------------------------------- | ------------------------- |
| 急速定义        | 连续同向 K 线运行，不允许出现回调（低点开始降低）       | 把“停顿”当成回调          |

| 止损优化        | 只在新重要波段点形成后再收紧                            | 中途随意“半路挪止损”      |

# 🧩 3. 核心知识点

## 3.1 建仓方案的“双重验证”

- 价格行为理据：解释为什么该方案在该环境下合理。
- 数学优势：证明长期统计为正期望（不是一次两次的好运）。

## 3.2 动能交易者为何在趋势里占优

- 趋势中反转尝试大多失败（课程给出 80%+ 的经验统计），形成惯性。
- 突破后常有第二推等距运动：可用实体高度投射与 AB=CD 测算目标。

# 🎯 4. 交易含义

- 入场：只在“突破有跟进 + 收线强劲 + 路径无明显阻力/磁体”时考虑收线追进。
- 风控：止损位置用“结构点”说话（推动起点/重要波段点），不要用情绪说话。
- 复盘：把每次追进标注为“急速阶段/通道阶段”，否则会混用逻辑。

## 2. 急速与窄幅通道的内涵与特征 05:05

### 1）急速 05:16

- ![](assets/太妃/img_28351160887150859.jpg)
- 定义特征：连续同向 K 线运行，不允许出现回调（低点降低），通常伴随突破行为。
- 结束信号：当出现首根回调 K 线且低点开始降低时，标志急速阶段结束，转入通道阶段。

### 2）窄幅通道 06:57

- ![](assets/太妃/img_3301737906618231765.jpg)
- 关键区别：与宽幅通道的根本差异在于逆势方能否通过限价/止损订单盈利。窄幅通道中逆势方仍难以获利，表明趋势动能保持良好。
- 时间框架关系：在更大时间级别图表上，窄幅通道常表现为急速运动，两者都应视为趋势延续阶段。

### 3）动能交易者的数学优势

- 自我强化机制：顺势方的数学优势与极速运动互为因果，形成"突破-追进-延续"的正反馈循环。
- 等距运动规律：突破后通常伴随第二推等距运动，包含实体高度投射和 AB=CD 两种测算方式。
- 统计优势：趋势中 80%以上的反转尝试会失败，源于动能交易者的集体行为形成的市场惯性。

### 4）收线市价追进方案 13:13

- BUYNOW 急速建仓方案

- ![](assets/太妃/img_2562059335807716286.jpg)

- 操作要点：

- 在每根趋势 K 线收线时市价追进

- 止损设于推动起点之外

- 目标位采用等距投射

- 适用条件：

- 突破点远离显著磁体

- 跟进 K 线收线强劲

- 路径无阻力障碍

### 5）应用案例 20:01

- 突破走势分析

- ![](assets/太妃/img_4138334524228398532.jpg)

- 成功特征：

- 突破后连续阴线无反弹

- 50%回调处出现阳线但未反转

- 等距目标与关键位重合

- 失败特征：

- 突破后立即出现长上影线

- 连续两次突破失败

- 形成三推高潮形态
- 止损管理原则

- ![](assets/太妃/img_6203695765313633822.jpg)

- 核心原则：止损必须基于价格行为理由（重要波段点），而非主观承受能力。

- 典型错误：中途随意调整止损缺乏理论依据，长期来看会破坏系统优势。
