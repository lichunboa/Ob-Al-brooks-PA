# 二、价格行为的光谱 00:25
- ![](assets/太妃/img_8724934253074724029.jpg)
- 市场动态本质：价格行为本质是多空双方在"均衡→失衡→再均衡"循环中的持续竞合过程
- 阿尔布鲁克斯方法论：读图时需要重点把握市场脉络与动能变化的深层规律
- 趋势阶段分析：

- 起始特征：突破原有平衡区间的确认信号

- 发展阶段：动能持续积累的通道形态演变

- 衰竭特征：趋势方突破力度减弱，逆势方开始出现盈利机会
- 交易筛选原则：建立市场循环框架后，应主动规避不符合当前阶段特征的交易机会
- ![](assets/太妃/img_5525858689193019530.jpg)
- 逆势交易信号：当逆势交易者开始出现持续盈利可能时，往往预示当前趋势可能进入末期
- 波段定位技巧：通过观察相邻波段高点/低点的相对位置关系，判断市场所处的循环阶段
- 三维分析维度：完整的价格分析需要同时明确"当前状态-来源路径-发展方向"三个维度

# 三、市场循环的定义 01:22
## 1. 市场循环与价格行为分析
- ![](assets/太妃/img_1228697445046664380.jpg)
- 基础概念回顾: 目前已学习突破、测试、回调、区间、反转等核心价格行为概念

### 1）趋势发展阶段分析
- ![](assets/太妃/img_5177399069123983181.jpg)
- 趋势启动特征:

- 连续出现强劲影线和短小阳线

- 初期维持清晰的上升节奏
- 回调测试机制:

- 首次深度回调会测试前期上升通道下沿

- 回调终点常触及关键波段支撑点
- 突破失败表现:

- 突破后立即折返形成假突破

- 引发逆势交易者更激进介入

- 产生比前波更急更强的回调
- 趋势延续信号:

- 关键低点未被有效跌破

- 趋势线虽被刺破但快速修复
- 趋势衰竭特征:

- 未能创新高后出现猛烈下跌

- 反弹力度逐渐减弱转为震荡

### 2）市场循环本质
- ![](assets/太妃/img_7711046007500704703.jpg)
- 循环定义:

- 从均衡到失衡再到新均衡的持续过程

- 包含趋势发起→削弱→震荡的完整周期
- 阶段划分:

- 极速窄幅通道阶段（强趋势）

- 宽幅通道阶段（趋势延续）

- 区间震荡阶段（均衡状态）
- 动能观察要点:

- 趋势方动能强弱变化

- 逆势交易者介入程度

### 3）交易警示
- ![](assets/太妃/img_3331400705668173656.jpg)
- 关键提醒:

- 在明显趋势中逆势交易"只会被止损到流眼泪"

- 必须通过细致价格行为分析判断市场阶段
- 分析要点:

- 识别当前处于四种阶段的哪一种

- 观察关键支撑/阻力测试结果

- 评估突破尝试的跟进力度

## 2. 市场循环的定义解析 03:32
### 1）趋势的四个阶段 03:46
- 急速阶段03:54

- 急速阶段的定义与特征

- ![](assets/太妃/img_2652420147017157982.jpg)

- 初始特征：表现为极速运动，回调幅度细小，整体上行斜率明确

- 演变过程：阳线连续性逐渐变差，比例从40%-60%逐步演变为70%-30%

- 通道变化：经历第二阶段窄幅通道后，回调幅度逐渐加深

- 急速阶段后的市场变化04:08

- ![](assets/太妃/img_3563320035034710133.jpg)

- 动能衰减：顺势方突破变得虚弱，最终演变为宽幅通道

- 状态转变：从单边交易逐步转向双边交易状态

- 最终形态：宽幅通道演变为交易区间，多空双方形成均衡

- 顺势交易者与逆势交易者的动态04:43

- ![](assets/太妃/img_1601912844399070098.jpg)

- 顺势方疲乏：随着价格升高，追高价值判断降低，开始提前止盈

- 逆势方活跃：空头积极进取，阴线数量增加，掌握更多K线

- 斜率变化：整体上行斜率逐渐趋缓，反映动能转换

- 急速阶段转向双边交易状态05:07

- ![](assets/太妃/img_5048907123195100258.jpg)

- 回调特征：回调次数增多、持续时间延长、深度加深

- 均衡形成：围绕公平价格形成宽幅区间，多空双方相互否决

- 反转信号：回调深度达到相互反转程度，标志趋势动能衰竭

- 市场循环的概念05:46

- ![](assets/太妃/img_3631879029943469674.jpg)

- 循环组成：由极速→窄幅通道→宽幅通道→区间四个阶段构成

- 非固定顺序：各阶段出现次数、顺序、持续时间均无定规

- 实际案例：可能出现宽幅通道直接突破，跳过某些阶段的情况

- 急速阶段的交易策略08:19

- ![](assets/太妃/img_4006515004601276204.jpg)

- K线特征：连续的趋势K线，实体饱满而引线短小，回调极少

- 最佳策略：仅进行顺势交易，避免逆势操作

- 风险收益：需承担较宽止损，但享有较高胜率和快速浮盈

- 介入时机：趁早介入可获得最佳效果，反映强烈的单边交易状态
- 顺势交易者占有极大数学优势09:57

- 机构类型10:07

- ![](assets/太妃/img_776263612458798426.jpg)

- 动能交易者10:17

- 行为特征：追涨杀跌，依靠趋势动能延续获利

- 决策逻辑：通过严格方法判定后，看到强劲突破即追进

- 算法特点：精明的机构行为，只在有利可图时建仓

- 价值交易者10:46

- 行为特征：等待回调，偏好更划算的建仓价格

- 策略差异：低处买入高处卖出，而非追高建仓

- 数学优势：在突破成功时，顺势方向几率可达60%以上

- ![](assets/太妃/img_2723929145845544832.jpg)

- 因果循环：数学优势与极速运动互为因果，形成自我强化

- 概率变化：趋势发起时顺势几率>60%，随持续时间逐渐降至<50%

- 机构行为：动能算法在正向期望时集中进场，推动趋势延续

- 均衡回归：过度运动后将折返，概率回归50%的均衡状态
- 回调阶段13:46

- 回调的启动进程13:47

- ![](assets/太妃/img_3579901097761763942.jpg)

- 动能交易者离场: 当价格靠近均衡位置时，保守型交易者开始离场，特别是动能交易者最先撤退

- 逆势方进场: 顺势交易者离场后，逆势方进场导致价格运动不确定性增加

- 价值交易者介入: 回调到一定程度时，寻求划算价格的价值交易者进场，促成新突破

- 急速状态结束标志: 回调K线形成时，上升趋势中K线低点开始降低，下降趋势中K线低点开始抬升

- 窄幅通道的阶段特征15:06

- ![](assets/太妃/img_8379897500423577835.jpg)

- K线组成: 主要由趋势K线构成，但长度可能稍短

- 回调特征: 出现更多短暂停顿和回调，回调K线长度不超过平均K线的两倍

- 逆势压力: 逆势交易者意愿增强但能力有限，造成的压力仍然很小

- 窄幅通道与极速的对比及交易策略16:16

- 时间级别差异: 在更大时间级别图表上，窄幅通道可能显示为极速；更小时间级别则可能显示极速为窄幅通道

- 交易策略:

- 顺势交易: 只进行顺势波段交易，避免逆势交易

- 常见误区: 不熟练交易者易将急速误判为高潮反转信号

- 持仓体验: 相比极速，顺势持仓体验稍差，浮盈增长较慢且伴有停顿回调

- 通道宽度与斜率的变化及意义17:26

- ![](assets/太妃/img_1217927388447978902.jpg)

- 广义通道概念: 所有价格运动都可视为某种通道，极速是宽度极窄、斜率极陡的特殊通道

- 通道演变观察点:

- 停顿和回调频率增加

- 回调持续时间延长

- 逆向运动深度加大

- 市场含义: 通道越宽，逆势交易者压力越大，顺势交易者对趋势控制力越弱

- 窄幅通道向均衡状态转变的迹象18:23

- ![](assets/太妃/img_9208642548328946982.jpg)

- 80%突破定理:

- 上升通道80%概率最终向下突破

- 下降通道80%概率最终向上突破

- 趋势强度判断:

- 强劲趋势回调短暂，逆势压力小

- 健康趋势表现为"形成旗形→突破→回调→再形成旗形"的循环

- 回调越深、对原趋势极值突破越弱，趋势动能越弱
- 宽幅通道阶段19:54

- 宽幅通道的特征20:00

- ![](assets/太妃/img_4302121523739393827.jpg)

- 回调特征: 回调明显增多且加深，持续时间可达5-10根K线以上

- 通道形态: 趋势线与通道线距离更远，价格运动连续性变差，整体斜率趋缓

- 市场状态: 已接近均衡状态，但仍保持一定趋势性

- 宽幅通道内的交易策略转变20:27

- 动能交易者退场: 追突破策略失效，进场后很快浮亏

- 价值交易者主导: 需要在回调中寻找划算入场点

- 盈利预期调整: 不应期待原趋势极值被有效突破

- 顺势交易者在宽幅通道中的策略20:46

- 入场时机: 在趋势线侧寻找入场点

- 持仓策略: 持仓至通道线侧即平仓，采取剥头皮交易方式

- 关键限制: 放弃突破预期，严格止盈

- 逆势交易者在宽幅通道中的机会21:44

- 盈利逻辑: 利用动能交易者缺席和市场共识进行逆势操作

- 操作方式: 在通道侧进场，目标为趋势线侧的利润

- 风险控制: 不宜押注突破趋势线，保持剥头皮交易风格

- 判断市场处于宽幅通道的重要性23:47

- ![](assets/太妃/img_4828730795417212424.jpg)

- 核心任务: 判断当前环境适合顺势还是逆势交易

- 难点: 介于极速和区间之间的模糊状态最难判断

- 关键问题:

- 顺势压力是否已消散？

- 逆势压力是否足以造成大型反转？

- 逆势交易者是否有盈利可能？

- 如何判断市场是否处于宽幅通道24:03

- 观察指标:

- 回调频率和深度明显增加

- 通道宽度显著扩大

- 斜率明显趋缓

- 行为验证:

- 突破追进策略失效

- 价值交易策略开始有效

- 逆势交易出现盈利机会
- 应用案例24:17

- 例题:趋势的起末切割阶段

- ![](assets/太妃/img_3706397632810115629.jpg)

- 趋势构成要素：一轮完整趋势由急速、窄幅通道、宽幅通道和区间四个阶段组成

- 动能演变特征：

- 急速阶段：多头动能强劲，表现为连续趋势K线

- 窄幅通道：开始出现短暂回调，但趋势K线仍占主导

- 宽幅通道：动能明显减弱，回调幅度和频率增加

- 区间阶段：多空力量均衡，价格横向震荡

- 判断技巧：

- 关键问句："逆势交易者当前能否轻松盈利？"

- 宽幅通道特征：逆势方可用限价订单轻松获取1倍盈亏比利润

- 窄幅通道特征：逆势方即使宽幅止损也难以持仓盈利
- 区间阶段27:58

- 例题:趋势所处阶段判断30:56

- ![](assets/太妃/img_4047539023732896679.jpg)

- 识别标准：

- 价格横向排列持续超过20根K线

- 回调幅度达到互相反转的程度

- 多空力量势均力敌，无明确趋势特征

- 交易策略：

- 在区间上1/3处做空，下1/3处做多

- 测算剥头皮利润，不押注突破

- 80%的区间突破会失败

- 形成原因：

- 趋势动能衰减至方向几率50%的均衡点

- 市场通过震荡寻找公平价格

- 可能演变为收敛三角形

- 例题:价格运动双边交易性格34:07

- ![](assets/太妃/img_7149134432671483737.jpg)

- 判断原则：

- 难以判断时默认按窄幅通道处理

- 真正的宽幅通道会明显显示逆势方优势

- 演变过程：

- 窄幅通道：斜率陡峭，回调短暂

- 宽幅通道：出现长阳突破趋势线

- 区间形成：建立双重底/顶结构

- 交易决策：

- 顺势交易更安全（如本例做空）

- 逆势交易需谨慎（如本例做多）

- 持仓不宜过远，及时止盈

- ![](assets/太妃/img_8676909255669169588.jpg)

- 关键指标：

- 回调持续时间3-10根K线

- 逆势压力逐步积累

- 80%通道会反向突破

# 四、市场循环的内容总结 38:13
## 1. 趋势四阶段理论
- ![](assets/太妃/img_4826371738378798266.jpg)
- 划分目的：将趋势起末切割成四个阶段，便于辨识和操作决策
- 核心价值：帮助交易者明确在不同阶段"该做什么，不该做什么"

## 2. 价格行为光谱理论
- ![](assets/太妃/img_367265325650953032.jpg)
- 光谱两端：

- 极端极速（极失衡状态）

- 极端区间（极均衡状态）
- 中间状态：统称为"通道"，其宽度反映市场特性

- 宽通道：双边交易强势（多空博弈激烈）

- 窄通道：单边交易强势（趋势明显）
- 循环本质：市场在失衡与均衡两个极端间持续摆荡
- 数学优势：顺势交易在趋势阶段具有60%-70%的数学优势

## 3. 动能与价值交易逻辑
- 动能交易："还能涨我就敢追"的追涨逻辑
- 价值交易："够划算我才建仓"的抄底逻辑
- 因果关系：顺势方的数学优势与急速运动形成互为因果

# 五、支撑阻力 40:26
## 1. 交易决策框架
- ![](assets/太妃/img_4337028709023865872.jpg)
- 核心问题：判断当前市场更有利于顺势还是逆势交易

- 市场脉络（整体趋势结构）

- 动能强度（价格运动能量）
- 逆向验证：通过"逆势交易者当前是否有盈利可能"反证趋势强度

## 2. 技术分析工具准备
- 磁体理论：支撑阻力的形成原理与标记方法
- 等距运动：市场价格波动的规律性特征
- 推动技术：趋势延续的识别与交易方法
- 学习路径：先系统掌握读图工具，再进入实战交易
