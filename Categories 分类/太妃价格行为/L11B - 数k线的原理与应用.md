# 🧠 1. 概览

- 推动/数K线的共同目标：用“多头/空头互相否决的次数”来量化多空力量对比。
- 市场惯性：顺势 2–3 推后更容易被逆势方阻断，随后进入回调与再平衡（偶见到 4 推）。
- 形态质量（匹配度）很关键：推动的间距、K线数量、振幅要“看起来匀称”，更易被机构算法识别并触发共识反应。
- 数K线是“更小尺度的数推”：在牛旗/区间底部数高点，在熊旗/区间顶部数低点。

# 🧩 3. 核心知识点

## 3.1 推动技术的“计分板”思维

- 把每次突破/拦阻当成一次“得分/失分”，让读图从“感觉”变成“次数口径”。
- 第二推结束常出现止盈与观望（AB=CD/等距目标等），这会改变动能结构。

## 3.2 匹配度：为什么“长得像”会更有效

- 形态越接近理想样貌，越容易被算法识别并形成群体反应。
- 若间距/振幅差异极大，可考虑把多个小推合并成一大推以保持结构匀称。

## 3.3 数K线：高点数法 / 低点数法

- 高点法关键是“先降后升”再抬高；连续抬高不计数。
- 低点法同理反向：必须“先升后降”再创新低。

# 🎯 4. 交易含义

- 止盈：第二推/第三推附近更容易出现“利润目标汇聚 + 顺势资金止盈”，优先做风控而不是盲目加仓。
- 预判反转：2–3 推后若未能继续突破，需提高对小型趋势反转/转区间的警惕。
- 过滤信号：匹配度差的“伪三推”不要硬做；先确保结构可被复述且视觉一致。

# 一、推动技术 00:02
## 1. 推动技术的本质功能
## 自动补充标题 00:20
- ![](assets/太妃/img_6878590475362806990.jpg)
- 核心功能: 通过记录多头突破空头次数和空头突破多头次数，帮助交易者清晰掌握市场多空力量对比
- 实际意义:

- 多头突破次数越多，代表买压积累越显著

- 空头突破次数越多，代表卖压积累越明显
- 市场惯性:

- 多数波段会在顺势2-3推后被逆势方阻断

- 通常1-2次阻断后会开启回调（偶尔会到4推）

- 波段内含2-3次顺势推动是常态现象

- ![](assets/太妃/img_8977008761665936113.jpg)
- 回调机制:

- 第二推结束时，交易者会通过AB=CD等方法测算利润目标并止盈

- 顺势交易者会转为观望，等待新的多空竞合结果

- 若趋势强劲，可能走出第三推尝试延续趋势
- 三推原理:

- 三推后若价格未突破，表明多空双方认为均衡不在该方向

- 此时价格会逆向探索更可能的均衡价位
- 应用场景:

- 可在2-3推后预判小型趋势反转

- 用于止盈或开仓决策

### 1）例题:突破后被拦阻 03:10
- ![](assets/太妃/img_3956261578191345638.jpg)
- 题目解析

- 案例展示开盘区间上破后的价格行为

- 突破后价格在等距运动目标位被反向订单拦阻

- 急坠下落中可能出现两种交易策略：

- 跟随空头动能追进

- 等待测试高点失败后下破瞬间入场

- 关键问题：空头如何确定合理止盈位？

- 可参考当日低点作为波段点

- 运用极速高度等距运动目标

- 当价格以第二、第三波段下行时：

- 需警惕歇行底形成的可能性

- 长下影线接粗绒阳线是离场信号

- 答案：结合磁体和数推技术，在多重不利因素汇聚处预判反转

## 2. 重视推动外观的匹配程度 08:28
### 1）匹配程度的概念 08:43
- ![](assets/太妃/img_3506542497771953671.jpg)
- 定义: 推动段落的规模（K线数量、振幅）和间距应保持视觉上的匀称
- 判断标准:

- 每段推动的K线数量和振幅相近

- 推动之间的时间间距不宜过大

- 整体呈现协调的视觉效果

### 2）匹配程度的重要性 09:32
- ![](assets/太妃/img_5837746069117898294.jpg)
- 市场机制:

- 多数机构交易者采用技术分析算法

- 形态越接近理想样貌，越容易被算法识别

- 识别度越高，市场参与者共同反应越强
- 实践意义:

- 匹配度差的形态可能被市场忽略

- 优质形态更可能按预期方式运作

### 3）匹配程度对市场预期的影响 10:06
- ![](assets/太妃/img_3484755891612673599.jpg)
- 负面案例:

- 若第一二推间距5根K线，第二三推间距130根K线

- 各推振幅差异显著

- 导致机构算法不视为同一组有效推动
- 结果影响:

- 减少反向订单部署

- 降低价格反弹概率

- 不值得冒险押注

### 4）如何判断推动外观的匹配度 11:18
- ![](assets/太妃/img_1706340868411218357.jpg)
- 三大要素:

- 间距：推动间的时间距离应相近

- K线数量：每推包含的K线数量应相当

- 振幅：价格波动幅度应保持比例协调
- 优化原则:

- 当小规模推动与大规模推动并存时

- 可将多个小推合并视为一大推

- 保持整体推动结构的匀称性
- 实践建议:

- 优先选择匹配度良好的推动组合

- 对不匹配的推动保持谨慎态度

## 3. 数K线 11:35
- ![](assets/太妃/img_8199297638052138435.jpg)
- 本质特征：与数推技术本质相同，都是计算多空相互否决次数的工具，区别仅在于观察规模更小
- 应用场景：主要适用于看涨环境（牛旗/区间底部）和看跌环境（熊旗/区间顶部）
- 常见误区：需避免三流交易者夸大的"奇效论"，本质上只是趋势分析工具

### 1）高点数法
- ![](assets/太妃/img_7049975190527345062.jpg)
- 高1定义：在牛旗或区间底部附近，高点高于前一根K线的K线
- 序列形成：

- 高2：高1后出现低点下降，再出现高点抬升的K线

- 高3/高4：第三次/第四次相同模式发生时
- 环境限定：必须在看涨环境下使用（上升趋势回调或区间下缘）
- 本质解读：标志空头反转向下尝试的第一推失败
- ![](assets/太妃/img_7862699299392563299.jpg)
- 典型环境：

- 上升趋势中的阴线回调

- 区间下缘（价格过度廉价区域）
- 操作要点：

- 需等待"先降低再抬高"的明确信号

- 连续抬高不计入序列（必须出现回调后再突破）

### 2）低点数法
- 对应关系：与高点数法原理相同，方向相反
- 低1定义：在看跌环境下，K线低点首次低于前一根K线
- 验证要求：需先出现低点抬升，再创新低才算低2
- 环境判断：适用于熊旗形态或区间顶部区域

### 3）数推关联性
- ![](assets/太妃/img_7540873633861103210.jpg)
- 规模对应：

- 高二回调 ≈ 二推回调 ≈ 双重底牛旗

- 高三回调 ≈ 三推回调 ≈ 蝎形底牛旗
- 共同原理：记录多空博弈次数（多头突破几次/空头突破几次）
- 反转预判：通常2-3推后可能反转，高4后未恢复趋势可能转为区间

### 4）例题:回调结束时机 16:24
- 题目解析

- 高1识别：阳线十字星后出现高点突破的阴线

- 高2确认：经历连续降低后再次出现高点抬升

- 形态对应：高二回调对应双重底牛旗形态

- 趋势恢复：高2成立后趋势延续

- 关键要点：必须严格遵循"先降后升"的计数原则

### 5）例题:三推运动识别 22:12
- 题目解析

- 环境判断：多空激烈博弈的宽幅区间交易日

- 低1定位：首次出现低点下降的K线

- 低2验证：经历抬升后再次创新低

- 蝎形特征：三次空头突破否决多头尝试

- 后续发展：完成三推后价格向区间中轴回归

- 经验提示：躁动交易日需结合更大级别图表分析

### 6）操作要点总结
- 环境选择：

- 看涨环境数高（牛旗/区间底部）

- 看跌环境数低（熊旗/区间顶部）
- 计数规则：

- 高点法仅关注高点重新抬高

- 低点法仅关注低点重新降低
- 规模认知：

- 高2≈二推回调

- 高3≈三推回调

- 高4≈区间转换信号
- 经验阈值：超过20根K线未恢复趋势则可能转为区间
