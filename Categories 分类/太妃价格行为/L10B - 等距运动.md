# 一、等距运动 00:03
## 1. 50%回调
## 自动补充标题 00:20
- ![](assets/太妃/img_4136211005721633294.jpg)
- 数学本质: 市场参与者基于精密数学逻辑产生的集体行为结果，是等距运动的变体形式
- 基本特征: 价格回调至前一波段上涨/下跌的50%位置时产生反转
- 形成原理: 价格距离两端极值（最高点和最低点）等距，代表多空双方在此价位达成交易平衡

## 2. 多空逻辑 02:24
- ![](assets/太妃/img_8877011403010577837.jpg)
- 多头策略:

- 入场点: 50%回调位

- 止损: 急速低点下方

- 止盈: 原趋势极值

- 盈亏比: 1:1
- 空头策略:

- 入场点: 50%回调位

- 止损: 原趋势极值上方

- 止盈: 急速起点

- 盈亏比: 1:1
- 趋势优势:

- 上升趋势中多头胜率可达60%

- 下降趋势中空头具有数学优势

- 通过趋势方向判断可提高交易胜率

## 3. 应用案例 04:20
### 1）例题:50%回调处买入
- ![](assets/太妃/img_6192913876541079885.jpg)
- 题目解析

- 案例特征: 强劲上涨后回调至50%位置

- 技术依据:

- 阳线动能饱满，卖压相对弱

- 趋势延续概率60%

- 操作逻辑: 在50%回调处挂限价买单

- 数学优势: 1:1盈亏比下具有60%胜率

### 2）例题:突破加50%回调 05:01
- ![](assets/太妃/img_3530531169872914301.jpg)
- 题目解析

- 复合因素:

- 对突破点的过冲测试(±50%\pm50\%±50%)

- 50%回调位支撑

- 验证方法: 测量前波段高低点距离

- 典型表现: 长下影线触及50%位置后回升

### 3）例题:双重底测试加50%回调 05:18
- ![](assets/太妃/img_7734716444021718921.jpg)
- 题目解析

- 形态特征:

- 前影线低点形成支撑

- 50%回调位验证支撑

- 交易逻辑: 双重验证提高入场可靠性

- 磁吸效应: 50%位置本身具有价格吸引力

## 4. 区间等距扩张 05:59
- ![](assets/太妃/img_2655522644774760030.jpg)
- 形态识别:

- 头肩顶轮廓

- 颈线突破后等距运动
- 交易信号:

- 突破颈线后空头入场

- 目标位=头部到颈线距离

## 5. 等距运动 06:17
- ![](assets/太妃/img_9050564155851997829.jpg)
- 反转确认要素:

- 趋势线突破

- 卖压充足的反转形态(如三推楔形顶)

- 对原趋势极值的测试

- 更缓趋势线被突破
- 50%回调应用:

- 下降趋势中在50%反弹位做空

- 需结合多重技术指标确认

## 6. 逐根K线阅读 07:50
- ![](assets/太妃/img_2331456451166495623.jpg)
- 常见误区: 多数交易者会选择性忽略看似无交易价值的K线，认为它们仅是机构算法产物或等待更清晰形态出现
- 成交量证据: 被忽略的K线与被交易的K线具有同等可观的成交量，证明市场活动始终存在
- 交易者义务: 必须理解每根K线传递的市场逻辑，这是成功交易的基础能力
- 学习特点: 耗时且困难，但不可回避，不能因难度而选择性忽略市场信息

## 7. 等距运动与50%回调在不同学派中的描述 08:17
- ![](assets/太妃/img_7187102136336326773.jpg)

### 1）形态学中的等距运动描述 08:38
- 经典案例: 头肩顶/杯柄形态突破后的等高运动
- 应用特点: 作为反复出现的现象被使用，但多数人不探究其数学原理
- 典型态度: 只关注"是否有效"，不关心"为什么有效"

### 2）费波纳契交易者的50%回调应用 09:07
- 特殊现象: 将50%回调(0.5)与非标准斐波那契数(0.618/0.382/0.236等)混用
- 效力确认: 实际交易中确实有效，但缺乏理论解释
- 理论缺陷: 0.5并非严格斐波那契数，但被纳入交易体系使用

### 3）亚当理论的镜像映射与等距原理 09:54
- 操作方式: 对已完成行情进行纵向+横向镜像翻转，预测未来走势
- 数学本质: 实际对应AB=CD的等距运动原理
- 解释局限: 使用者通常无法说明镜像预测的理论依据

### 4）各学派对于原理的探究态度 10:41
- 普遍现象: 多数交易者停留在"发现现象-使用现象"层面
- 卓越要求: 需要理解数学逻辑和价值/动能交易者意图
- 方法论价值: 只有理解原理才能在方法失效时进行诊断和改进
- 核心警示: "学而不思则罔"——避免盲目使用而不求甚解的危险
- ![](assets/太妃/img_834065287285027891.jpg)

## 8. 磁体的总整理与综合演练 11:22
- ![](assets/太妃/img_2268468887050794124.jpg)
- 磁体分类：实用的磁体有五大类，包括前波段高/低点（趋势极值、重要波段点）、区间上下缘、趋势线与通道线、均线（如EMA20\mathrm { E M A } 2 0EMA20）、等距运动目标与50%回撤位。
- 动态标记技巧：在读图过程中需动态标记这些潜在转折点，以清晰跟踪价格脉动和多空力量对比。
- 应用选择：五类磁体都应关注，但实际开仓时可侧重水平磁体（如区间边界、50%回撤位），不同交易者可发展个人偏好的磁体使用风格。

### 1）例题：K线拆解基础 12:34
- ![](assets/太妃/img_3905132210241828746.jpg)
- 核心问题：价格逐根生成时，需反射性思考哪些磁体导致价格缓速或折返。例如：

- 大阳线后出现包覆十字星形成"铁丝网"区间

- 阴线突破失败时，阳线高点成为阻力位

- 价格反弹位置常与均线、等距扩张目标重叠
- 复合磁体案例：

- 50%回撤位与均线重叠时易引发趋势反转

- 头肩顶形态突破后，目标位常与AB=CD等距运动复合

### 2）例题：进阶K线拆解 19:04
- ![](assets/太妃/img_8882269282061418082.jpg)
- 典型场景分析：

- 长下影线止步于上涨50%回撤位，显示多头在此关键位置介入

- 均线阻抗现象：EMA20\mathrm { E M A } 2 0EMA20多次阻止价格穿越

- 通道线重叠：前区中点同时是通道AB=CD目标和重要波段点
- 突破后确认：

- 成功下破区间低点后，原高点应标记为重要波段点

- 头肩顶形态下破后需确认回调高点位置
- 多空博弈逻辑：

- 多头在50%回撤位介入的数学优势

- 空头尝试构建双重顶压制价格

- 真空效应导致价格加速测试磁体后急转直下
- 复杂通道处理：

- 扩张下降通道中，趋势线连接端点可明确方向

- 失败突破仍会产生等距运动（如空头止损位引发反向买单聚集）

- 将震荡区视为区间或收敛通道的两种标记方法
- 实战要点：

- 二推AB=CD回调常终结于窄幅通道50%位

- 均线阻抗与突破点测试形成复合阻力

- 末端旗形可能预示大型趋势反转

- ![](assets/太妃/img_3081160645932775779.jpg)
- 核心机制：

- 价格波动本质是从一个磁体运动到另一个磁体的过程

- 突破瞬间最早入场者的止盈位会成为反向订单聚集点

- 无论突破方向如何，等距扩张原理均适用
- 训练建议：

- 每日图表阅读量决定交易水平（如30年阅读7560张图表案例）

- 需独立标记关键位置并理解每个转折的磁体作用原理
