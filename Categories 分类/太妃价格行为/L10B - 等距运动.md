# 📺 1. 课程概览 (Module Overview)

> **本节核心 (Core Theme)**：`50% 回调`可以看作等距运动的一种“均衡版本”。它之所以能反复出现，是因为在中点附近，多空双方用相似的止损/止盈结构（常见 1:1）去争夺，市场在此更容易形成短期平衡并发生折返。

---

# 🧾 2. 速查表 (Cheat Sheet)

| 场景 | 多头/空头常见计划（简化） | 备注 |
|---|---|---|
| 50% 回调做多（上升趋势） | 入场：50%位；止损：急速低点外；止盈：原趋势极值；RRR≈1:1 | 体系经验：顺趋势胜率可到约 60% |
| 50% 回调做空（下降趋势） | 入场：50%位；止损：原趋势极值外；止盈：急速起点；RRR≈1:1 | 关键是“顺趋势”加持 |
| 复合磁体 | 50%位 + 突破点测试/均线/双重底等叠加 | 叠加越多，入场理由越充分 |

---

# 📝 3. 核心知识点拆解 (Key Topics)

## 3.1 50% 回调的数学/均衡含义
- 价格在前段高低点的中间位置，距离两端极值等距。
- 这往往对应多空在此形成“交易平衡”（更容易出现折返）。

## 3.2 多空逻辑：为什么 1:1 也能有优势
- 在趋势背景下，顺势一方的胜率可上升（原文口径：约 60%）。
- 用“趋势方向”过滤，能够把 50% 回调从纯 50/50 的均衡里拉出优势。

## 3.3 复合磁体：把概率堆起来
- 50% 位如果同时是：突破点过冲测试、双重底/顶验证、均线重叠等，会显著提高“到位反应”的可信度。

## 3.4 跨学派共性：同一规律，不同叫法
- 形态学：头肩顶等距扩张目标。
- 斐波那契交易：把 0.5 与 0.618/0.382 混用（0.5 不是严格斐波那契数，但实操有效）。
- 亚当理论：镜像映射，本质对应 AB=CD 等距逻辑。

## 3.5 读图方法论：逐根K线看“磁体驱动”
- 每根K线都在表达多空博弈与订单作用；不能只挑“看起来有形态”的K线。
- 训练重点：动态标记五类磁体，并解释每次减速/折返对应的磁体。

# 🎯 4. 交易含义 (Trading Implications)

- 50% 回调更像“均衡交易”：必须依赖背景（趋势/磁体叠加）来提升胜率。
- 看到复合磁体时，优先关注“到位后的反应”（影线、收盘位置、跟进），而不是只看是否触及。
