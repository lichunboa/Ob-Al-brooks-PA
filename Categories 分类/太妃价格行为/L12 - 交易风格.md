# 🧠 1. 概览

- “交易风格”不是审美偏好，而是对 $胜率\,(p)$ 与 $盈亏比\,(R)$ 取舍后的系统选择。
- 市场不存在“高胜率 + 高盈亏比”的长期稳定机会：你看到的优势，往往意味着对手方出现不合理选择。
- 波段 vs 剥头皮的核心差异：持仓时间、容忍回撤/止损频率、以及对执行速度的要求。

# ⚡ 2. 速查表

| 风格            | 典型盈亏比 |     典型胜率 | 典型持仓         | 更适合                       |
| --------------- | ---------: | -----------: | ---------------- | ---------------------------- |
| 波段（Swing）   |     ≥ 2.0R | ≈ 40% 或更低 | 约 10–30 根 K 线 | 耐心型、能忍受频繁止损       |
| 剥头皮（Scalp） |     ≤ 1.0R |        ≥ 60% | 约 1–5 根 K 线   | 敏捷型、厌恶浮亏、偏快进快出 |

> 期望值口径（示例）：$E = p\cdot R - (1-p)\cdot 1$。例如 $R=2,\,p=0.4$ 与 $R=1,\,p=0.6$ 的期望都可相近。

# 🧩 3. 核心知识点

## 3.1 为什么“高胜率 + 高盈亏比”很难同时成立

- 若你长期拿到高胜率又高盈亏比，意味着对手方在做低胜率又低盈亏比的决策；在机构参与的市场里通常不现实。
- 更常见的是：高盈亏比策略用低胜率换取“大盈利尾部”；高胜率策略用低盈亏比换取“稳定小利润”。

## 3.2 新手路径：先稳定，再扩展

- 优先专精一种风格（课程建议偏向先做波段），每天只挑 1–2 笔高质量机会。
- 用交易日志与回测验证：盈亏比 > 1.0 且胜率 ≥ 50% 才能越过盈亏平衡线。

# 🎯 4. 交易含义

- 避免“风格错配”：波段交易者在区间里强做突破、剥头皮者在趋势里逆势抓反转，常把优势变成劣势。
- 风格选择以个性为先：能忍受浮亏/踏空更适合波段；能承受高频决策与小利润更适合剥头皮。
- 先把一套风格做成“可重复的正期望”，再谈融合。

### 6）疑问解答 26:50

- 盈亏比与止盈止损的关系

- ![](assets/太妃/img_6371645564563086432.jpg)

- 直觉冲突现象：单纯用盈亏比划分波段/剥头皮交易时可能出现反直觉情况，例如：

- 盈亏比 2.0-2.3 倍，但止盈仅几根 K 线宽度

- 宽止损宽止盈的 1.0 倍盈亏比交易，本质是更大时间级别的剥头皮

- 窄止损窄止盈的 2.0 倍盈亏比交易，本质是更小时间级别的波段

- 常规标准：

- 波段：四成胜率+两倍盈亏比+持续 10 根 K 线以上

- 剥头皮：高胜率+1.0 倍盈亏比+1-5 根 K 线持仓
