# 📺 1. 课程概览 (Module Overview)

> **本节核心 (Core Theme)**：反转本质上是“逆势方的突破成功 + 顺势方的突破失败”，但在趋势初/中期，绝大多数反转尝试会失败（约 80%）。因此交易的重点不是“看到反向就猜换边”，而是识别：这是小型趋势反转（更像回调/旗形），还是具备条件的大型趋势反转。

> [!INFO] 本节的 3 个关键问句
> 1) 当前看到的是小型趋势反转（回调）还是大型趋势反转？
> 2) 是否已经出现关键结构：趋势线突破、关键点失守（前低/前高）、以及对原趋势极值的测试？
> 3) 多空动能对比是否发生了“从单边到双边”的稳定转变？

|---|---|---|

# 📝 3. 核心知识点拆解 (Key Topics)

## 3.1 反转的工作定义
- 反转 = 顺势方突破失败 + 逆势方突破成功 → 运动方向改变。
- 微观上可以在小级别看到：上升趋势中低点降低/下降趋势中高点抬升。

## 3.2 小型 vs 大型趋势反转
- 小型：局部方向改变，最终多转化为持续形态（旗形/回调）。
- 大型：改变主要趋势方向，通常需要更长时间（10–20 根以上）与更充分的压力积累。

## 3.3 趋势健在时“反转尝试的倾向性”
- 在趋势初期和中期，约 80% 的反转尝试会失败。
- 关键心法：不要把“任何反向波动”都当成换边信号；要等结构与跟进。

## 3.4 大型反转的常见前置结构
- 趋势线突破（第一信号）。
- 防线被击破（多头前低失守 / 空头前高抬升）。
- 对原趋势极值的测试：不及/持平/过冲会影响后续走向（延续 vs 区间）。

# 🎯 4. 交易含义 (Trading Implications)

- 趋势初/中期：默认反转尝试更可能失败，优先按回调/旗形思维处理。
- 只有当“结构 + 跟进”都出现，才逐步提高反转成立的权重：趋势线突破只是开始。
- 观察多空动能对比：从单边 → 双边的转换是否稳定（阴阳线数量/实体变化/收盘位置/影线信息）。
