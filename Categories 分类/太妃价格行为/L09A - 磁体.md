# 📺 1. 课程概览 (Module Overview)

> **本节核心 (Core Theme)**：所谓“磁体”，本质就是支撑/阻力（潜在反向订单的密集区）。价格经常被这些关键价位“吸引”去测试；越接近，路径中订单越稀薄（真空效应）时越容易加速。

---

# 🧾 2. 速查表 (Cheat Sheet)

| 概念 | 一句话 | 交易上的用法 |
|---|---|---|
| 支撑 | 下方潜在买单密集区 | 触及前不能确定会反弹，需看反应 |
| 阻力 | 上方潜在卖单密集区 | 触及前不能确定会回落，需看反应 |
| 磁体 | 支撑/阻力的“到站点” | 预判下一个测试目标位（到站逻辑） |
| 真空效应 | 途中订单稀薄 → 加速 | 趋势强时别逆势挂限价单“拦车” |
| 突破结果 | 成功约 20%，失败约 80% | 失败更常见：回归均衡；成功则趋势加速 |

---

# 📝 3. 核心知识点拆解 (Key Topics)

## 3.1 支撑/阻力 = 潜在订单
- 支撑：价格下方潜在买入订单聚集；接近时空头止盈（买入）+ 多头进场。
- 阻力：价格上方潜在卖出订单聚集；接近时空头建仓 + 多头止盈（卖出）。
- 关键点：都是“潜在拐点”，触及前无法确认是否有效。

## 3.2 磁体视角：价格在磁体之间往返
- 市场经常在支撑/阻力之间震荡测试，寻找“公平价格”。
- 在区间里：突破尝试约 80% 失败（维持现状的惰性）。

## 3.3 真空效应：越靠近磁体越容易加速
- 交易者倾向在关键位“以逸待劳”挂单，中途拦截订单稀薄。
- 于是出现：缓慢靠近 → 突然加速的非线性运动。

## 3.4 实操纪律：接近磁体时怎么做
- 磁体附近优先顺当前方向（避免被加速段“真空效应”打脸）。
- 不要在磁体处直接用逆势限价单赌反转；等看到明确反应/形态再处理。

- 磁体只能提示“目标位”，不保证“路径与耗时”，必须做风险管理。

- 支撑与阻力的定义 02:13

- ![](assets/太妃/img_6798316472148776222.jpg)

- 支撑定义：在当前市场价格下方潜在积聚大量买入订单的价位。当价格下行接近时，空头止盈（相当于买入）与多头进场形成合力，导致买盘突然超过卖盘引发反弹。

- 阻力定义：在当前市场价格上方潜在积聚大量卖出订单的价位。当价格上涨接近时，空头建仓与多头止盈（相当于卖出）形成合力，导致卖盘突然超过买盘引发回落。

- 关键特征：两者都是"潜在"的变盘点（Inflection Points），价格触及前无法确知是否真会引发转向。
- 支撑与阻力的形成原因

05:22

- ![](assets/太妃/img_9010251944317143216.jpg)

- 两大形成逻辑：

- 利害关系：交易者为解套（如突破后错误方向持仓者寻求盈亏平衡）在特定价位部署订单

- 心理预期：交易者基于历史价格行为（如区间边界、前高低点）预判多空力量变化

- 市场验证标准：一个价位需有60%以上概率产生至少可供剥头皮交易的折返幅度，才能被认定为有效支撑/阻力

- 唯一实质：反向订单的集中部署是支撑阻力存在的唯一原因，没有订单聚集的价位只是"空气"
- 支撑与阻力的作用方式

12:11

- ![](assets/太妃/img_4647092964956205349.jpg)

- 基本作用模式：

- 测试：价格到达关键价位时多空重新交战

- 突破结果：

- 成功突破（20%概率）：趋势加速，被套/踏空者加入顺势方向

- 突破失败（80%概率）：价格折返，市场回归均衡状态

- 趋势中的特殊性：

- 健康趋势中多数支撑/阻力会被突破，仅趋势末期可能出现有效反转

- 疲弱的突破尝试往往是趋势反转的预警信号

- 交易应用：

- 顺趋势交易需观察价格击穿支撑/阻力的力度

- 逆势交易需在区间边界结合突破失败概率部署

- 磁体本质：支撑/阻力位如同磁铁吸引价格反复测试

- 区间内80%突破尝试失败

- 需建立"价格到站"思维，预判下一个可能测试的支撑/阻力位

- 价格运动规律: 价格始终在支撑阻力间折返往复，像被磁铁吸引般趋近目标位置，形成"到站-折返"的运动模式。

- 典型表现: 常见"缓慢靠近→突然加速"的价格运动形态，如2021年比特币69000美元高点前后的波动。

20:10

- 形成机制: 交易者"以逸待劳"心理导致价格路径上反向订单稀薄（如区间边线聚集大量订单而中途稀少）。

- 第二推现象: 在区间突破时，由于多数交易者选择在边线入场，造成价格加速冲向磁体（如标普500指数2022年4800点测试）。

- 订单分布: 支撑阻力位实质是潜在反向订单的密集区，真空状态使价格穿透中间区域时阻力减小。

- ![](assets/太妃/img_8105276552234498120.jpg)

- 动态过程:

- 趋近阶段：价格进入磁力范围受牵引加速（真空效应主导）

- 接触阶段：磁力消失导致价格转向或穿透（订单消耗完毕）

- 持续运动: 价格总会寻找下一个磁体目标，形成市场波动的内在驱动力。

### 3）磁体的注意事项 24:03
- 价格运行目的地

24:14

- ![](assets/太妃/img_5292877051825898296.jpg)

- 预测局限:

- 仅能指示可能目标位（如台湾加权指数18650点必会回测）

- 无法预判具体路径（直线反弹或多次探底）和耗时（数月或数年）

- 交易警示:

- 不可因"必然回测"而忽视资金管理

- 临近磁体比远期目标更具实操价值
- 价格触碰磁体前交易原则

26:21

- ![](assets/太妃/img_8329711795961249916.jpg)

- 方向选择:

- 在磁体附近只顺当前方向建仓（避免逆势被真空效应杀伤）

- 典型案例：2022年1月标普500接近4800时不宜过早做空

- 订单部署:

- 禁止直接在磁体挂限价单逆势入场（尤其在趋势强劲时）

- 首次测试成功率不确定，需观察实际反应
- 价格到位前判断原则

26:52

- 动态评估:

- 需持续监控多空力量对比变化

- 注意途中可能出现的可靠建仓形态（如双重底/顶）

- 风险控制:

- 必须设置合理止损（避免被曲折路径洗出）

- 不因"终将到达"而忽视短期波动风险
- 价格触碰磁体时判断原则

27:36

- 关键观察点:

- 动能变化：延续/停顿/反转（如比特币测试前高时的量价配合）

- 突破概率评估：需综合考量趋势强度、测试次数等因素

- 决策窗口:

- 价格触及后反应速度极快，需提前制定应对方案

- 成功突破则跟踪新磁体，失败反转则捕捉回归机会
