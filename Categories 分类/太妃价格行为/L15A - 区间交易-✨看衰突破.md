# 🧠 1. 概览

- 区间（Trading Range）= 双边交易性格主导：阴阳线交错、重叠多、买卖压力充分，胜率围绕 50% 附近摆动。
- “看衰突破（look to fade）”= 你不看好趋势 K 线能形成有效突破，于是在区间里做相反方向交易，押注突破失败后的折返。
- 关键前提：必须真的是区间（通常要足够“久/大”）；并且你获利方向要有足够空间，否则就算突破失败也不划算。

# ⚡ 2. 速查表

| 项目     | 你在找什么                                                               | 常见错误                           |
| -------- | ------------------------------------------------------------------------ | ---------------------------------- |
| 区间成立 | 持续时间足够（例如日内 80 根 K 里，>20 根 K 更像区间）+ 上下边界反复测试 | 把趋势里的浅回调当区间，逆势“送钱” |
| 区间大小 | 区间高度足够，能给出合理盈亏比                                           | 区间太窄（铁丝网），折返也赚不到   |
| 失败证据 | 看到边界附近出现突破失败（过冲/持平/不及都可）                           | 只凭感觉“这里该折返了”             |
| 止损     | 放在区间边线之外（常见 1–2 个最小报价单位，按品种波动调整）              | 止损太宽换虚高胜率，实际盈亏比差   |
| 止盈     | 倾向区间中轴附近（约 33%–66%）或按风险偏好微调                           | 贪到另一侧边线，反而提高回撤       |

# 🎯 3. 交易含义（区间交易 + 看衰突破）

- 适用场景：价格在区间内反复测试边界但难以走出趋势；你更愿意与突破“对赌”，赚取失败后的均衡回归。
- 入场逻辑：
  - 先确认区间（时间/规模/上下边界测试）
  - 等待边界附近的“突破失败线索”
  - 再在折返迹象出现时介入（而不是第一次触碰边线就进）
- 风险收益：
  - 止损在边线外、止盈在中轴附近，本质是在控制回撤的前提下赚“回到均衡”的一段。
  - 如果区间过窄，你会陷入“必须押注另一边突破才赚钱”的矛盾结构，应放弃。

# 一、区间交易方法 00:09

## 1. 区间

## 自动补充标题 03:22

- ![](assets/太妃/img_5295178977633239138.jpg)

### 1）区间的定义与特点

- 核心特征: 指双边交易性格占主导地位的市场段落，表现为阴阳 K 线交错重叠、买卖压力相对充分的状态
- 形成原因: 趋势末尾常陷入区间是因为随着远离失衡点，顺势交易胜率从 60−70%60-70\%60−70%降至 50%50\%50%，引发逆势方进场
- 时间分布: 市场 80%时间处于这种不起眼的双边交易状态，失衡导致的趋势运动仅占少数时间

### 2）区间与趋势的关系 03:45

- 循环关系: 失衡启动趋势 → 快速移动到新均衡 → 形成区间 → 新的失衡突破区间 → 开始下一轮趋势
- 运动特征: 趋势运动在图表上醒目但时间短，区间状态持续时间更长但价格波动幅度较小

### 3）区间与通道的关系 04:25

- 宽幅通道本质: 虽具趋势性格，但逆势方可盈利，属于双边交易地带
- 判断标准: 通道由窄变宽过程中，区间特性越明显；当斜率接近消失时，康复通道与区间几乎无异
- 关键区别: 完全均衡的区间应水平无斜率，表示多空双方均无顺势优势

### 4）骑行回调也是区间 05:15

- ![](assets/太妃/img_3653769024737018847.jpg)
- 典型表现: 蝎形底/牛级形态中，多空相互吞没、斜率平缓、突破屡屡失败的段落
- 特殊类型: 趋势中的逆向宽幅通道（如旗形回调），本质仍是双边交易强烈的地带

### 5）区间突破与趋势延续的解析 05:47

- 矛盾现象: 区间倾向突破失败，但趋势中的骑行回调最终却能延续原趋势
- 脉络解释:

- 上升趋势中的牛企回调，卖出力量来自多头止盈和空头剥头皮

- 买入力量来自踏空多头和被套空头回补

- 宏观趋势脉络使区间最终倾向顺势突破
- 局部表现: 在最终突破前，区间上缘仍会体现突破失败的拦阻特征

## 2. 区间特性

- ![](assets/太妃/img_4797384749011462205.jpg)

### 1）突破失败倾向

- 关键参数:

- K 线数量：20 根为分界（基于 80 根 K 线的日内交易习惯）

- 回调深度：超过前趋势运动 75%时应视为显著区间
- 规模影响:

- 小型区间在趋势中作为短暂整理，最终倾向顺势突破

- 大型区间（K 线多、高度大）形成宽泛均衡，突破难度大

### 2）价格折返机制

- 均衡本质: 多空围绕 50%胜率中轴震荡

- 价格向上偏离：空头占优

- 价格向下偏离：多头占优
- 边界形成:

- 下缘：多头认为划算而积极买入，推动价格折返

- 上缘：做空优势引发空头大举入场，形成反向订单积聚

### 3）突破后行情发展

- 等距运动: 突破后倾向走出基于区间高度的等距运动（AB=CD 形态）
- 两推理论:

- 第一推：动能交易者基于 60%60\%60%胜率逻辑，目标为区间高度等距投射

- 第二推：价值交易者修正踏空/被套仓位，推动接续运动
- 循环重启: 价格远离区间标志新失衡开始，进入下一市场循环

## 3. 区间的特性 07:17

### 1）看衰突破成功 12:13

- ![](assets/太妃/img_2358023650144820092.jpg)
- 术语来源: 源自阿尔布鲁克斯交易体系，原文为"look to fade"，简中译本译为"淡出趋势 K 线"
- 核心概念: 指不看好趋势 K 线能够造就突破，于是进行相反方向的交易，预期突破失败而盈利
- 行为本质: 相当于与突破做对，通过反向订单"投下反对票"
- 适用场景: 主要应用于区间交易中，当观察到价格在区间内反复测试边界但未能形成趋势时
- ![](assets/太妃/img_5839381055700734475.jpg)
- 空间要求: 获利方向必须有充足空间，区间高度不能太局限

- 若区间过窄（如铁丝网形态），即使突破失败也难以获得合理盈亏比

- 需避免出现"必须押注价格向另一边突破成功才能获利"的矛盾局面
- 确认条件: 需要有充分证据表明价格确实处在区间内

- 若实际处于趋势中，逆向压注会变成"送钱给对手方"

- 需结合第六单元区间界定方法进行判断

- ![](assets/太妃/img_1041076211499623521.jpg)
- 持续时间: 对于 80 根 K 线的日内交易，区间应持续超过 20 根 K 线以上
- 突破失败证据: 需同时具备对原趋势极值和第一推回调极值的突破失败

- 突破失败形式不限（过冲、持平或不及均可）

- 建议初学者采用较保守的确认标准
- 边界划定: 通常以四点（上下各两点）一线方式确定区间边界
- 例题：突破后等距运动 16:32

- ![](assets/太妃/img_2203813482451700939.jpg)

- 题目解析

- 形态特征: 倾斜向下的三推斜形底，低点不断递进但都以失败告终

- 交易行为: 大阳线吸引多头入场，但下一根 K 线未跟进导致多头被套

- 后续发展: 连续下跌触发止损，演变为对失败牛旗通道线的突破

- 等距运动: 突破后形成基于形态高度的等距运动，测试突破点后走出第二推

- 关键观察: 阳线放大急涨后出现怠速上行，最终形成散推先行点进入回调

- ![](assets/太妃/img_2724612336054257985.jpg)

- 题目解析

- 回调特征: 回调测试隐含 K 线低点和 50%回调处

- 区间条件: 对原极值和回调极值双双突破失败，区间条件完备

- 边界锚定: 可采用最外侧顶点作为区间边线价位

- 交易信号: 保守双重底交易者可能在此触发买入信号

- 入场时机: 明确做区间交易者应等待再次测试边线后折返迹象才入场
- 例题：突破成功跟进 20:06

- ![](assets/太妃/img_1674145538570984037.jpg)

- 题目解析

- 止损设置: 通常放在区间边线之外 1-2 个最小报价单位

- 越过信号 K 线高点到第二个报价单位才真正触发

- 不同品种需按波幅比例调整

- 止盈设置: 建议在区间中轴附近（区间宽度的 33%-66%）

- 中轴为多空均衡地带，越过则不利于原方向

- 可根据风险偏好调整具体位置

- 设置原理: 避免使用过宽止损换取虚高胜率，应保持合理盈亏比

- ![](assets/太妃/img_757842138860928334.jpg)

- 题目解析

- 形态演变: 窄幅区间有效下破后形成三推楔形底，成就等距运动

- 交易机会: 等待对原趋势和回调极值的双双突破失败

- 边界确认: 最保守者要求两边都出现突破失败，居中者至少要求一边

- 二次入场: 首次止损后可考虑是否介入第二次交易机会

- 市场特性: 案例印证市场更倾向于维持均衡状态而非突破
