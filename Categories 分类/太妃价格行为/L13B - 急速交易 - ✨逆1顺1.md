# 🧠 1. 概览

- 核心思路：趋势有延续性；在缺少强磁体阻力/动能未衰竭时，“逆势第一次反转（逆1）”更常失败，而“顺势第一次恢复（顺1）”更常成功。
- 这套逻辑与急速/窄幅通道相容：突破越远、回调越浅、路径越通畅，顺势方越占优。
- 两种测算口径：无回调时更偏“实体投影等距”；有回调时更偏“AB=CD（C点=回调最深处）”。

# ⚡ 2. 速查表

| 要素 | 好的样子 | 坏的样子 |
|---|---|---|
| 突破要远 | 价格持续远离突破点，跟进阳线饱满 | 突破幅度递减，动能衰退 |
| 测试要浅 | 回调短浅、很快恢复趋势 | 回调深入威胁突破点 |
| 路径通畅 | 突破方向无显著磁体/波段点拦阻 | 路径有磁体，易提前折返 |
| 止损位置 | 放在急速起点之外（结构点） | 随意“半路挪止损” |

# 🎯 3. 交易含义

- 入场：等待逆1失败后的顺1信号（例如：信号K线出现后，高点先降低再抬升，再用停止单触发）。
- 目标：若有回调，用 AB=CD 更保守；若无回调，实体投影更贴近动能逻辑。
- 过滤：若“突破不远 + 回调很深 + 路径有磁体”，宁可放弃（或把目标降到磁体处）。

# 一、急速建仓方案之二 00:03
## 1. 趋势性向延续 00:27
### 1）逆1失败、顺1成功建仓方案

- ![](assets/太妃/img_3290464727187560385.jpg)
- 核心原理：趋势具有延续性，除非遇到明确磁体阻力导致动能衰竭，否则逆势反转尝试通常会暂时失败
- 操作特征：
    - 在上升趋势中，空头首次反转尝试（逆1）会失败
    - 多头恢复趋势的高一尝试（顺1）将会成功
- 市场机制：
    - 每次突破会套牢逆势交易者，同时吸引踏空的顺势交易者入场
    - 趋势初期顺势动能最强，逆1往往被自救操作迅速终结
    - 第一推之后通常伴随第二推的等距运动

    - 市价追进者需提前判断动能

## 7. 突破的方向上没有磁体造成拦阻 08:40

- ![](assets/太妃/img_5487077825180425888.jpg)
- 入场前控管要点: 持仓方向上的路径应保持通畅，避免存在磁体阻挡价格运行
- 磁体影响机制: 当价格运行路径上存在波段点等磁体时，可能导致价格未达目标就折返，造成利润回吐
- 应对策略:
    - 保守做法：发现磁体存在时应考虑放弃交易
    - 激进做法：可将目标位调整至磁体所在位置，降低胜率要求
    - 最佳情况：路径完全无磁体阻挡，此时潜在反向订单最少

## 8. 应用案例 09:37

### 1）例题:突破幅度要求

- ![](assets/太妃/img_6250421404082221225.jpg)
- 案例对比要素:
    - 突破动能：右边交易日表现为更饱满的趋势阳线，突破更远
    - 测试表现：右边价格回调后立即大幅跃起，左边则持续阴线扩大下跌
    - 通道特征：右边形成窄幅陡峭通道，左边呈现宽幅通道特征
- 决策要点:
    - 突破后跟进的基本要求需达成
    - 对动能要求高的交易者可选择放弃弱势突破日
    - 急速追进时应设置最低突破幅度标准
- 失败信号识别:
    - 当价格回测突破点并极致突破失败时
    - 多头动能疲软表现否定原建仓理由
    - 需及时将宽幅通道视为不利因素离场

### 2）例题:趋势突破交易 13:17

- ![](assets/太妃/img_3618485996268707107.jpg)
- 优质交易三大要素:
    - 突破动能：越强劲越好，突破点距离越远越佳
    - 测试表现：回调动能越弱越好，折返幅度越浅越佳
    - 路径通畅：突破方向上无磁体阻挡
- 个性化调整:
    - 激进型：可接受较高风险，设置较宽松标准
    - 保守型：需严格筛选，宁愿踏空也不冒险
    - 需平衡踏空与止损的取舍关系
- 实际应用示范:
    - 开盘前标记重要磁体位置（如区间极值、波段高点）
    - 突破确认后测算等距运动时需预判潜在卖单位置
    - 阳线连续出现时可考虑试驾追进，但目标测算应保守
    - ![](assets/太妃/img_978875313756587502.jpg)
- 磁体应对方案:
    - 尊重磁体所在位置，不武断预测其必被突破
    - 可选择在重要波段点提前止盈
    - 最远目标不应超过区间极值
- 成功案例特征:
    - 突破距离足够远（要素一）
    - 回调测试较浅（要素二）
    - 维持陡峭窄幅通道上行
    - 最终使收线市价追进和逆一顺一方案都获利
